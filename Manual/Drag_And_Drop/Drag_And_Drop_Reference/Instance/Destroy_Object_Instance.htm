销毁对象实例

只要你想 "摧毁 "一个实例，你就可以调用这个动作，这将把它从房间中移除。你应该注意，当你摧毁一个实例时，它的Destroy事件会在调用Destroy的动作后立即被调用。还要注意的是，尽管Destroy事件是在动作被调用的瞬间执行的，但该实例并没有立即从游戏中移除，它将继续执行 它将继续执行当前事件中包含的代码，直到结束，只有当当前事件结束时，实例才会被从游戏中移除，这时将调用清理事件。
