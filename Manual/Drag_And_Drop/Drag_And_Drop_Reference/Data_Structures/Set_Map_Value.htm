设置地图值

通过这个动作，你可以在地图数据结构中添加一个键/值对。你提供了持有地图索引的变量（由动作Create Map返回），然后给出一个 "键"（这是地图中一个值的标识符 然后给出一个 "键"（这是一个值在地图中的标识符），然后是与该键相关的 "值"。两个键都可以是整数值（如0、200、36712等）或字符串（如 "姓名"、"盾牌 "等），但值可以是任何 有效的数据类型。
