插入到列表中

通过这个动作，你可以恢复一个项目在给定列表中的索引位置。你提供存储列表索引的变量（由动作Create List返回）和你想在列表中找到的索引位置的值。 以及一个目标变量来存储返回的项目索引位置（可以标记为临时局部变量，直到脚本或事件结束）。当你向列表中插入一个值时，所有的索引 都会向上移动一个，所以如果你有一个包含 4 个项目的列表（索引位置 0 到 3），并在索引位置 2 放置一个新的项目，之前的项目 2 会变成项目 3，之前的项目 3 会变成项目 4。 请注意，如果你要找的值在列表中不存在，那么将没有列表索引位置可以返回，所以返回值将是-1。
