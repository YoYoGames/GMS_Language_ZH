如果数据结构为空

通过这个动作，你可以检查一个给定的数据结构是否为空。你首先需要选择要检查的数据结构类型（List、Map或Stack--没有网格，因为它们甚至不能是空的），然后你提供一个 变量来保存数据结构的索引（如创建列表、创建地图和创建堆栈等动作所返回的）。如果数据结构是空的，该动作将返回true，如果是假的，则返回false。 是空的，如果不是，则返回false。注意，你可以在动作中标记 "not"，通过检查一个数据结构是否为空，来检查它是否持有数据。
