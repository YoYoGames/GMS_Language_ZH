绘制变换后的值

通过这个动作，你可以在屏幕上画一个字符串标题和/或一个变量或值。标题参数是一个你希望加在数值前面的字符串，如果需要可以留空，而数值可以是一个变量、一个固定值或 另一个字符串--在这种情况下，它应该用引号括起来，如果需要，也可以留空。然后，你给出字符串应该被绘制的位置，可以是一个绝对位置，也可以是相对于绘制 文本的相对位置。此外，你还可以设置文本的比例，沿着水平轴或垂直轴拉伸它，以及旋转（旋转文本的角度是逆时针的，所以设置为90°将向左旋转，-90°向右旋转 右）。)缩放是按给定的系数进行的，所以0.5的缩放将使文本的宽度减少一半，2的缩放将使宽度增加一倍，1是不缩放的默认值。请注意，文本在绘制时的效果将取决于 使用的字体、颜色和对齐方式。所有这些都可以在绘制前改变，如下面的例子所示。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，将不会抽到任何东西。

yscale y（垂直）比例因子，默认为1
