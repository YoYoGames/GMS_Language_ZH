绘图值

通过这个动作，你可以在屏幕上画一个字符串标题和/或一个变量或值。标题参数是一个字符串，你想把它加在值的前面，如果需要的话，可以留空，但如果你想包括文本的话，应该用引号把它包起来。 值可以是一个变量、一个固定值或另一个字符串--在这种情况下它也应该用引号括起来--但不能留空。然后，你要给出字符串应该被绘制的位置，可以是一个绝对位置 或相对于绘制文本的实例的位置。注意，文本在绘制时的外观将取决于所使用的字体、颜色和对齐方式。所有这些都可以在绘制前改变，如下面的例子所示。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
