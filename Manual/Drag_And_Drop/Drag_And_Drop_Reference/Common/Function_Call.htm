功能调用

这个动作可以用来调用任何GML函数或任何用户定义的函数。你给出要调用的函数，然后提供它所需要的任何额外的 参数，即：你希望输入到函数中的任何值（字符串、整数、资源指数等......），以便它能够使用它们。你可以通过点击下面的加号图标来增加额外的参数输入 旁边的加号图标 你可以通过点击动作旁边的加号图标来添加额外的参数输入，你应该确保你的参数数量与正在使用的函数相匹配（即：如果函数需要3个参数，你应该提供3个参数）。一些函数也可以返回一个值，以便以后在动作块中使用 列表中使用，因此你也可以指定存储该返回值的目标变量，该变量可以是之前创建的，也可以是一个新的临时本地变量（如果你选中 "Temp "复选框）。目标 和参数栏都可以留空，如果它们不是必需的。
