设置粒子的形状

这个动作可以用来设置粒子在绘制时应该使用的内置精灵形状。你提供要设置形状的类型的唯一ID值（正如你用动作Create Particle Type创建该类型时返回的那样 和要使用的形状。粒子形状只是为GameMaker Studio 2创建的白色精灵，专门用作粒子，由以下形式组成。

注意：如果你使用这个动作，那么你就不应该使用Set Particle Sprite这个动作，因为你只能有一个精灵或一个形状，但不能同时拥有。
