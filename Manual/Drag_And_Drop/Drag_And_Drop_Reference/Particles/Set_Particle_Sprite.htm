设置粒子精灵

这个动作可以用来设置粒子在绘制时可以使用的精灵。你提供要设置精灵的类型的唯一ID值（就像你用动作Create Particle Type创建该类型时返回的那样 和要使用的精灵资源。如果精灵有子图像，那么粒子将以图像编辑器中为精灵设置的速度在各帧中产生动画。还要注意的是，如果你为粒子设置了一个颜色值（使用Set Particle Colour动作），那么这个颜色将与所使用的精灵混合，就像在实例上设置图像混合将与实例精灵混合一个颜色一样（使用白色表示不混合）。

注意：如果你使用这个动作，那么你就不应该使用设置粒子形状这个动作，因为你只能有一个精灵或一个形状，但不能同时拥有。
