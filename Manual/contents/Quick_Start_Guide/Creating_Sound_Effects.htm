创造声音效果

所有的游戏都要有声音效果--大多数也有音乐--这意味着你需要能够在GameMaker Studio 2中把它们添加到你的游戏中。这可以通过 声音编辑器, 来实现的，当你在资产浏览器中创建一个新的声音资产时，它就会被打开。

GameMaker Studio 2中的声音可以是未压缩的 ，也可以是压缩的，压缩的声音在运行时有额外的选项来处理它们。未压缩的声音是那些被保存为 WAV 格式的文件，而压缩的声音可以是 MP3 或 OGG 格式。一般来说，你会希望小的声音效果是 WAV - 像射击声，或按钮声。 等--而较大的效果或音乐则是 MP3 或 OGG 。与其他不同的资源，它建议你使用一个适当的前缀命名你的声音，如" snd_" 这样你以后就可以很容易地识别它。

注意：未压缩的声音只是意味着它没有经过任何类型的压缩。 算法的压缩。对声音进行压缩意味着它需要更少的存储空间，但也意味着它在播放前需要解压，因为小的声音效果通常使用得很多，而且非常快--并不占用存储空间 由于小的声音效果通常使用得很多，而且非常快--不占存储空间--你不希望在播放时有额外的CPU成本来解压它。然而，由于像音乐文件这样的东西通常是非常大的文件，你确实希望它们被压缩，因为它们会占用更少的存储空间。 而解压的额外CPU成本也可以忽略不计，因为它们一般只是一次播放一个。

添加声音就像点击添加声音按钮一样简单， ，然后在打开的文件资源管理器中浏览你想使用的文件（你也可以把声音文件拖到 释放声音文件到GameMaker Studio 2IDE上，它们将被添加为新的声音资产）。)一旦你选择了一个并加载了它，你就可以使用显示的回放控制来预览它，并改变某些属性，以确定该文件应如何导出。 文件应该如何导出的某些属性。如果你不确定这一点，那么只需保持一切 "原样"，但如果你正在添加音乐或 MP3 / OGG 格式的声音，你可能应该改变 到压缩-非流格式。

请注意，声音会占用磁盘的大量空间，所以在为你的游戏项目创建声音时，你应该小心。你真的需要一个48KHz的声音效果，只播放一秒钟的时间吗？可能不需要!所以，考虑把这里的采样率降低到11025或22050的效果，对于像音乐这样长的东西或循环效果，也要考虑使用压缩格式。还要注意的是，如果你打算使用3D定位的音频效果。 那么最初的声音文件必须是单声道的。
