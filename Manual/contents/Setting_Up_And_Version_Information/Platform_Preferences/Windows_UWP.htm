Windows UWP首选项

Windows UWP偏好有以下选项。

  • Visual Studio位置。该选项用于将GameMaker Studio 2指向构建PC上的MicrosoftVisual Studio安装。
  • Win UWP工具的位置。该选项用于将GameMaker Studio 2指向构建PC上的UWP工具安装路径。
  • 默认的软件包架构。默认情况下，在编译UWP可执行文件时，你会看到一个对话窗口来选择x86、x64 和ARM 架构。你可以把这个选项 到这三个中的一个，然后当你选择编译一个可执行文件时，就会默认选择这个选项。
  • 自动响应创建存储包：在编译时显示的对话窗口中，你可以选择创建一个存储包。你可以把这个选项设置为"是 "或 "否 "，它将在你选择编译可执行文件时被默认使用。 每当你选择编译一个可执行文件时，都会默认使用。
