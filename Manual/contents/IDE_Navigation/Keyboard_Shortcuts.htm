键盘快捷键

GameMaker Studio 2有大量的键盘快捷键，虽然大部分都在IDE中与之相关的菜单项旁边做了标记，但并不是所有的快捷键都是或在一开始就能让用户看出来。在这里你可以找到 所有键盘快捷键的完整列表，以方便参考指南。一般来说，快捷键可分为以下三类。

  • 全局- 全局快捷键是那些无论你在GameMaker Studio 2IDE中的哪个位置都可以使用的快捷键，除非你的焦点在一个特定的窗口或小工具上，而这个窗口或小工具会覆盖它。
  • 窗口 - 窗口快捷方式是那些特定于单一窗口或标签的快捷方式，除非有小工具快捷方式覆盖它。
  • 小工具- 小工具的快捷方式非常特别，只有在使用单一的小工具（例如，颜色选择器）时才能发挥作用。

下面是完整的快捷键列表，根据编辑的不同分成几个部分。
