设置网格值

通过这个动作，你可以将网格数据结构中的一个特定单元设置为一个值。你提供了一个保存网格索引的变量（由动作Create Grid返回），然后给要设置的单元格的x和y位置（列 和行），然后给出要设置的单元格的x和y位置，最后是该单元格的值。值可以是任何有效的数据类型，但x和y位置必须是网格范围内的整数。

行动语法。

争论。

争论 描述
栅格 要设置的网格的索引（存储在一个变量中）。
x 要设置的单元格沿X轴的位置（列）。
y 要设置的单元格沿y轴的位置（行）。
价值 将网格单元设置为的值
