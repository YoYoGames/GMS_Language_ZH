设置重力方向

这个动作是用来设置内置重力变量的方向。每个实例都可以有自己的重力效应，如果重力值被设置为零以外的任何值，那么这个实例将被 "拉 "向 这个动作设置的方向。你可以在0°和360°之间设置一个方向，其中右为0°，上为90°，左为180°，下为270°。请注意，你可以将其设置为相对于当前的重力值，所以无论你输入什么，都会被加上/减去 加/减。
