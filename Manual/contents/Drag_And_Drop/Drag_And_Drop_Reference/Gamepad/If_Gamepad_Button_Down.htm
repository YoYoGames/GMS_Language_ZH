如果游戏手柄按钮向下

通过这个动作，你可以对游戏板进行轮询，看一个给定的按钮目前是否被按住了。这个动作在按钮被按下的每一步都会返回 "true"，否则返回 "false"，不过如果你点击not修改器 你可以检查按钮是否被按住，即：当没有按钮被按住时，动作将返回true，如果有的话，则返回false。如果你只需要检查单个按钮的按下情况，那么就使用If Gamepad Button Pressed动作。

如果被检查的按钮是一个模拟按钮，那么直到按钮的值超过了用动作Set Gamepad Button Threshold设置的阈值，它才会被认为被按住。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为真，这些动作现在将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
