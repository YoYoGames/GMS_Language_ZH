设置视图变量

这个动作允许你设置某些与视口如何显示房间的摄像机视图有关的内置变量。你选择要编辑的视口变量，然后给视口以目标（从0到7），最后给变量的新值 变量的新值。视口是屏幕上用于绘制特定摄像机视图的区域（如在房间编辑器中设置的），你能够以这种方式设置的不同变量在下表中列出。

变化的 描述
Camera 这是用来设置视口以显示不同的相机视图。你需要提供要转换的摄像机的唯一ID值（更多信息请看这里 更多信息）。)
Visibility 这是视图端口的可见性切换，期望值为true或false，设置为true将使视图端口可见（即：它将绘制到屏幕上），设置为false将不绘制任何东西到屏幕上。 绘制到屏幕上。
View Port X Coordinate 这是游戏窗口中视口的X位置。请注意，只有在多个视口被激活时才真正需要它--例如创造一个分屏效果--如果在只有一个视口被激活时使用，你会得到 会得到意想不到的结果。
View Port Y Coordinate 这是游戏窗口中视口的Y位置。但请注意，只有在多个视口处于活动状态时才真正需要它--例如创造一个分屏效果--如果在只有一个视口处于活动状态时使用它，你会得到 会得到意想不到的结果。
View Port Width 这是视口的宽度（单位：像素）。视图端口的宽度（如果有多个视图端口被激活，则为组合视图端口）定义了游戏开始时的游戏窗口或背景画布的宽度。 游戏开始后改变这个值将不会对游戏窗口的大小产生可见的影响，除非与GML函数一起调用 window_set_size(). 还要注意的是，如果你的端口尺寸比分配给摄像机的尺寸大或小，那么摄像机的视图将被缩小--或放大--以适应。
View Port Height 这是视口的高度（单位：像素）。视图端口的高度（如果有多个视图端口被激活，则为组合视图端口）定义了游戏开始时的游戏窗口或背景画布的高度。 游戏开始后改变这个值将不会对游戏窗口的大小产生明显的影响，除非与GML函数一起调用 window_set_size(). 还要注意的是，如果你的端口尺寸大于或小于分配给摄像机视图的尺寸，那么摄像机视图将被缩小--或放大--以适应。
View Port Surface ID 通过这个变量，您可以设置将某个视口的内容绘制到一个曲面上，您提供的值是用于绘制视口的曲面。更多信息请看这里。
