设置健康

通过这个动作，你可以为健康状况创建或设置一个实例变量。你可以提供一个值来设置健康状况变量或添加到健康状况变量中（如果你勾选相对复选框，那么该值将被添加到当前的 "健康 "值）。)请注意，如果你想使用绘制实例健康条动作，那么这个值应该在0到100之间，否则它可以是你需要的任何数量。还要注意的是，与 常规的实例变量，"健康 "的变量名是内部存储的，所以要检索分数值，你需要使用Get Health动作。
