创建对象实例

通过这个动作，你可以从资产浏览器中创建一个任何对象的实例。你要给出对象的名称和它在房间里的X/Y位置，以及要创建实例的层的名称（必须是一个 实例层）。)你可以勾选 "相对 "标志，以确定实例相对于运行动作代码块的实例的位置。每当你在游戏中创建一个实例，它就会被赋予一个唯一的ID值，然后你可以用它来 识别该实例，因此这个动作允许你定义一个 "目标 "变量来保存这个ID值（如下例所示）。请注意，目标变量可以被标记为一个临时（本地）变量，它将为动作创建，并且可以被删除。 为该动作创建，并可在所有后续动作中使用。

争论。

宗旨 要创建一个实例的对象的名称。
x 在房间中创建实例的X位置（可以是相对于调用实例的位置）。
y 在房间中创建实例的y位置（可以是相对于调用实例的位置）。
层数 要创建实例的实例层的名称（一个字符串）。
目标 作为返回值的目标变量。
