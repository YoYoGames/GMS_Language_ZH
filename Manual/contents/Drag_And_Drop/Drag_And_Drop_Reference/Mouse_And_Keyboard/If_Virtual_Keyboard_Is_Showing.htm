如果虚拟键盘显示出来了

这个动作可以用来检查操作系统的虚拟键盘是否被显示。当键盘开放或打开时，该动作将评估为真，如果虚拟键盘关闭或关闭，则评估为假。如果你启用了not修改器，这个动作将被反转，你可以将返回的真/假值存储在一个目标变量中，该变量可以被标记为temp（本地）。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为 "true"，这些操作现在将被运行。 评估为 "真"时，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
