如果文件存在

通过这个动作，你可以检查一个给定名称的文件是否存在。你选择要检查的文件（作为一个刺，包括扩展名），如果它存在，该函数将返回真，如果不存在则返回假。需要注意的是 如果该文件作为一个包含的文件存在于游戏包中，或者如果该文件在游戏运行时被保存出来，该函数将返回true（更多信息见文件系统部分）。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
