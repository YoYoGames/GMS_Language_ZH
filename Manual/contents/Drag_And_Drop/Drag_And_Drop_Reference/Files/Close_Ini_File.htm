关闭Ini文件

这个动作将关闭当前打开的Ini文件。这个动作应该在你完成对任何打开的Ini文件的读写时调用（Ini文件是用打开Ini文件这个动作打开的）。如果你在使用了写到Ini文件的动作后没有使用这个动作，那么就不会有任何东西被写入磁盘，因为文件信息被保留在内存中，直到这个动作被调用，这时它就会从内存中取出来 并写入磁盘上的文件。还要注意的是，如果你试图打开一个Ini文件，而之前没有关闭另一个文件（或同一文件），你会得到一个错误。
