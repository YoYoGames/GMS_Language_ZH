打开Ini文件

通过这个动作，你可以打开一个Ini文件进行读取和/或写入。如果Ini文件在你检查的位置不存在，GameMaker Studio 2可能会创建一个，但只有当你向它写入数据时才会创建。如果你只从Ini文件中读取信息 Ini文件，那么读函数的默认值将被返回，但Ini文件实际上不会被创建。
