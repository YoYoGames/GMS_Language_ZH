设置文本对齐方式

这个动作将为所有后续的绘制文本动作设置字体对齐。你可以设置水平对齐方式为左、中或右，垂直对齐方式为上、中或下，文字将相对于绘制动作的x/y 的位置对齐，如下面的图片所示。

文本向左水平对齐
文本在水平方向上与中心对齐
文本在水平方向上向右对齐
文本垂直于顶部对齐
文本垂直于中间对齐
文本在底部垂直对齐

请注意，这将设置所有后续绘制文本调用的对齐方式，因此可以在游戏开始时调用一次，例如在任何情况下，而不仅仅是在绘制事件中，所有绘制文本的动作都将使用设置的对齐方式。 然而，如果你要求多个绘制文本的动作以不同的方式对齐，那么你需要在每个你希望以不同方式对齐的绘制文本动作之前调用这个动作。
