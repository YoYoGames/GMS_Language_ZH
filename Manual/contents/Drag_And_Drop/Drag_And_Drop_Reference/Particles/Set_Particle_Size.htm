定的颗粒大小

这个动作用于设置粒子的大小，它可以是静态的，也可以设置为随时间增加/减少。你为要设置尺寸的类型提供唯一的ID值（正如你用动作Create Particle Type创建该类型时返回的那样 然后给出一个初始尺寸范围，然后是增加量，最后是 "摆动"。粒子被创建的大小是一个标量值，其中1是与基本粒子图像相同的大小，0.5是基本图像的一半，而2则是与基本图像相同的大小。 0.5将是基本图像的一半，2的值将是两倍。如果最小和最大的尺寸不一样，那么创建的任何粒子的初始尺寸将是它们之间的一个随机值。

增加粒子大小的选项可以让你输入一个值，这个值会在粒子寿命的每一游戏帧中添加到基本标度值上。因此，将这个值设置为，例如，0.1，将在每个游戏帧上增加0.1的比例值。 请注意，你也可以在这里使用负值，使粒子图像的比例也下降。请注意，可以接受的最小缩减值是-0.01，最小增量是0.01。

最后你可以设置 "摇摆 "选项。这是一个将在0和你给的值之间随机产生的值，然后在粒子寿命的每一步上增加或减少尺寸，使绘制的图像在尺寸上波动 大小。很明显，大的数值比小的更明显，这个数值甚至可以是负数，最大范围在-20和20之间。
