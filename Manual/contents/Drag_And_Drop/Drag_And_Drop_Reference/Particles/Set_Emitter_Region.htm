设置发射器区域

这个动作用于设置一个粒子系统中发射器的区域和形状。你提供发射器所属的粒子系统ID（由动作Create Particle System返回）和 实际的发射器ID（由动作Create Particle Emitter返回），然后在给定的边界框值内给出一个发射器的形状。可能的发射器形状是。

为了定义你的形状，你要给出一个 "边界框 "的坐标，给定的形状将被放入该框中（如果是矩形、椭圆和菱形形状），或者是起点和终点的第一和第二坐标（如果是直线）。 形状的坐标（如果是矩形、椭圆和钻石形状），或者是起点和终点的第一和第二坐标（如果是直线）。所有从发射器中迸发的粒子都将在这个形状的范围内创建。 在这个边界框内的形状范围内创建。你还可以通过以下选项来设置粒子的分布。

  • 线性。粒子将在给定的形状内的任何地方随机创建。
  • 高斯型。粒子将主要在形状的中间产生，而沿边缘的粒子较少。
  • 反高斯。粒子将主要沿着形状的边缘创建，中间的粒子较少。
