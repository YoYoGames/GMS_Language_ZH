设置粒子阿尔法

这个动作将为你创建粒子时设置alpha，以及在其生命周期内设置alpha。你首先指定粒子类型的ID值（这应该与你使用创建粒子类型动作创建粒子时返回的值相同），然后你可以设置要使用的alpha值。

你可以选择三个阿尔法值来使用，而粒子将在粒子的生命周期中在它们之间混合。例如，如果一个粒子的阿尔法值是0.5、1和0，并且有60个游戏帧的寿命，那么当它被创建时（第0帧 它的阿尔法值是0.5，然后它将渐渐变成完整的阿尔法值1（第30帧），然后再渐渐变成看不见的阿尔法值0（第60帧）。
