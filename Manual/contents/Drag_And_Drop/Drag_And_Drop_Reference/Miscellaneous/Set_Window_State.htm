设置窗口状态

通过这个动作，你可以将游戏窗口的状态设置为全屏或窗口化。这里有三个选项。

  • 切换- 当选择这个选项时，每次你调用动作时都会在全屏和视窗状态之间切换窗口。
  • Fullscreen- 这将使窗口从窗口状态进入全屏状态。
  • Windowed- 如果是全屏，这将使窗口返回到窗口状态。

请注意，游戏在全屏状态下的外观取决于你的目标平台的游戏选项--图形设置，因为如果游戏没有一些机制来调整 如果游戏没有一些机制来调整不同的长宽比和分辨率，你会得到拉伸（如果游戏选项被设置为全比例），或者你会得到黑条（如果游戏选项被设置为保持长宽比）。一般来说，如果你没有任何措施来处理不同的分辨率，你应该 处理不同的分辨率，你应该对你想编译的目标平台使用保持高宽比。
