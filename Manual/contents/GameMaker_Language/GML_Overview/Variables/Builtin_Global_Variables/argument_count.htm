论据数

这个只读变量返回传递给脚本函数或方法的 "参数 "的数量。通常与 一个参数数组 (参数[0...15])一起使用，以允许一个给定的函数有不同的输入参数。

注意：这不再是严格意义上的要求，更多的是为了支持传统的东西而提供。所有用户定义的函数将接受变量参数，任何传入函数的参数如果不属于命名的 输入的任何参数都将被默认初始化为未定义，这可以通过is_undefined()函数来检查。

语法。

argument_count;

返回。

整数

例子。

函数 my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argument[i]
}
else
{
arg[i] = -1;
}
}
}

上面的代码使用argument_count变量以及 argument 数组来初始化变量，是一个允许用户定义的函数接受可变数量参数的例子。

返回。全局变量
