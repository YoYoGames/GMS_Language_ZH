异步加载

这个变量是全局范围的，在异步事件中使用时，用来保存DS地图。 和-1在所有其他时间。DS地图的实际内容将取决于异步事件回调的类型，以及用于触发事件的函数，所以请参考 关于所有可能的DS地图内容的完整细节，请参阅这些事件的单独页面。

语法。

async_load;

返回。

实数（整数值

扩展实例。

sprite = sprite_add("site.com/path/image.png", 16, true, true, 0, 0)。

上述代码将在一个事件中被调用，从外部URL加载一个精灵。然后，这将触发图像加载异步事件，在那里你将解析 async_load 地图。

if ds_map_find_value(async_load, "id") == sprite
{
if (ds_map_find_value(async_load, "status") >= 0)
{
sprite_index = sprite;
}
}

上面的代码将首先检查async_load地图的ID，然后检查回调的状态。如果该值大于或等于0（标志着成功），那么回调的结果将被用来设置该 实例的精灵索引设置为新加载的图像。

返回。全局变量
