绘制圆角

通过这个函数，你可以画一个圆角矩形的轮廓，或者画一个填充的圆角矩形，其中（x1,y1）位置是左上角，（x2,y2）位置是右下角。你可以用函数定义画圆角的精确程度 角的精确程度。 draw_set_circle_precision()来定义角的精确程度，但这个函数使用了一个固定的半径（如果你需要改变角的半径，可以使用函数 draw_roundrect_ext()). 请注意，由于各种支持的平台之间的差异，被绘制的矩形可能需要不同的值（在x、y、或宽度或高度上的+/-1）来绘制所需的尺寸。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_roundrect(x1, y1, x2, y2, outline)。

争论 描述
x1 圆角矩形左上角的X坐标（就像一个普通的矩形一样）。
y1 圆角矩形左上角的y坐标（就像一个普通的矩形一样）。
x2 圆角矩形的右下角的X坐标（就像一个普通的矩形一样）。
y2 圆角矩形右下角的y坐标（就像一个普通的矩形一样）。
纲要 圆角矩形是否被填充（false）或作为一个像素宽的轮廓（true）绘制。

返回。

不适用

例子。

draw_set_colour(c_lime);
draw_roundrect（50, 100, 250, 200, 0）。

这将画出一个填充的圆角矩形，其理论上（在被弯曲之前）的左上角在（50,100），其理论上的右下角在（250,200）。
