gpu_get_tex_max_aniso

在使用tf_anisotropic过滤模式时，通过这个函数你可以得到当前最大的各向异性水平（见 gpu_get_tex_mip_filter()了解更多信息）。)返回的值将 范围在1到16之间。

语法。

gpu_get_tex_max_aniso（）。

返回。

实数（默认：16

例子。

if gpu_get_tex_max_aniso() != 8
{
gpu_set_tex_max_aniso(8);
}

上述代码将检查当前的最大各向异性过滤水平，如果不是8，就设置为8。

返回。Mipmapping
