顶点格式化_添加正常

告诉GameMaker Studio 2接受表面法线数据（nx、ny和nz）作为正在创建的新顶点格式的一部分。

语法。

vertex_format_add_normal()。

返回。

不适用

例子。

vertex_format_begin();
vertex_format_add_textcoord()。
vertex_format_add_normal();
my_format = vertex_format_end();

上面的代码将创建一个只有纹理和表面法线值的新顶点格式，然后将格式ID存储在变量 "my_format "中。

返回。基元和顶点格式
