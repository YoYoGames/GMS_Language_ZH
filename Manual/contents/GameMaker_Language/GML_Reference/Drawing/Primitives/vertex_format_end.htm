顶点格式_结束

这个函数必须在定义了任何新的顶点格式之后被调用。它返回新格式的 "句柄"（索引），这个句柄必须在所有进一步引用这个新格式的顶点函数中使用。

语法。

vertex_format_end()。

返回。

真实的

例子。

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上面的代码将创建一个只有颜色和位置值的新顶点格式，然后将格式ID存储在变量 "my_format "中。

返回。基元和顶点格式
