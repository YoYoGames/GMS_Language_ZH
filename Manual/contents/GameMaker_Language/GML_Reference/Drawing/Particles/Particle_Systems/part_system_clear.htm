体系的清空

通过这个函数，你可以将索引系统清除到其默认状态，删除所有的发射器并重新设置深度和位置（如果它们被改变了）。使用这个函数要小心，因为如果你有任何实例设置或爆裂或任何其他的 动作，你将会得到一个错误，除非你用这个 part_emitter_exists()函数来检查。也没有必要调用 part_emitter_destroy()函数 因为这也是自动处理的。

注意：这个函数将清除房间里的可见粒子，但它不会清除粒子的属性，也不会从内存中删除它们。为此，你应该使用函数 part_type_clear() 和 part_type_destroy().

语法。

part_system_clear(ind)。

争论 描述
ind 要清除的粒子系统的索引。

返回。

不适用

例子。

part_system_clear(global.Sname)。

上述代码将清除全局变量 "Sname "中索引的粒子系统，使其处于默认状态。
