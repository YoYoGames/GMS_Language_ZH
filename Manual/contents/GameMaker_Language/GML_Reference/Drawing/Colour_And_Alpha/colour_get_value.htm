color_get_value

这个函数将返回给定颜色的值（发光度）。这是混合到最终颜色中的 "光 "的数量，是定义颜色的色调、饱和度和数值方法的一部分。下面的图片说明了 这个值是如何对应于HSV色阶的。

语法。

color_get_value(col)。

争论 描述
殖民地 要检查的颜色

返回。

沟通

例子。

col = make_colour_hsv(random(255), 255, color_get_value(c_teal))。

上面的代码得到了用于制作颜色常数 "c_teal "的值，然后用这个值来设置一个随机的颜色，使其具有相同的亮度，将得到的颜色存储在变量 "col "中。
