绘画_获取像素

通过这个函数，你可以得到正在绘制到当前渲染目标的任何像素的颜色值。这意味着结果将取决于调用该函数的事件，也取决于正在使用的目标表面。需要注意的是 这个函数不会返回任何alpha值，为此你应该使用 draw_getpixel_ext().

注意：这个功能将导致巨大的性能冲击，因此只有在绝对必要的时候才可以使用。

语法。

draw_getpixel(x, y);

争论 描述
x 要检查的像素的X坐标
y 要检查的像素的y坐标

返回。

真实的

例子。

draw_set_colour(draw_getpixel(500,200))。

上述代码将把所有文本、表格和基元的绘图颜色设置为游戏屏幕上(500,200)处像素的颜色。
