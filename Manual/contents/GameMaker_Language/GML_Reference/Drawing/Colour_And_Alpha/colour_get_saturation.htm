颜色_获取饱和度

这个函数将返回给定颜色的饱和度。这是混合到最终颜色中的色调的量，是定义颜色的色相、饱和度和值（亮度）方法的一部分。下面的图片说明了 这个值是如何对应于HSV色阶的。

语法。

color_get_saturation(col)。

争论 描述
殖民地 要检查的颜色

返回。

沟通

例子。

col = make_colour_hsv(random(255), color_get_sat(c_teal), 255);

上面的代码得到了用于制作颜色常量 "c_teal "的饱和度，然后用这个值来设置一个随机的颜色，使其具有相同的饱和度，将得到的颜色存储在变量 "col "中。
