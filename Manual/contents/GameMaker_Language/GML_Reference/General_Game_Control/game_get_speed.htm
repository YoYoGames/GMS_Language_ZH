游戏_获取速度

这个函数可以用来获取游戏速度，可以是每秒运行的游戏帧数，也可以是每个游戏帧的微秒数。请注意，这不是实际的运行速度FPS值（为此使用 fps_real变量）。) 而是游戏每秒试图维持的游戏帧数（FPS），或者游戏试图维持的每一帧的长度（MPF）。当你使用这个函数时，你需要给出以下其中一个 常量，这将决定返回值的类型。

恒定 描述
游戏速度_fps 用每秒的帧数来获取游戏速度。
速率_microseconds 用每帧微秒来获取游戏速度。

因此，例如，如果游戏选项中的游戏速度设置为30，而你使用FPS类型，那么该函数将返回30，但如果你使用MPF，那么该函数将返回33333。

语法。

game_get_speed(type)。

争论 描述
类型 用来获得游戏速度的方法类型（见上面的常数）。

返回。

真实的

例子。

if game_get_speed(gamespeed_fps) != 60
{
game_set_speed(60, gamespeed_fps);
}

上面的代码检查游戏是否以60FPS的游戏速度运行，如果不是，就设置为60FPS。
