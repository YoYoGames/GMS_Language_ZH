显示领导板的成就

这个函数将打开所选平台的排行榜页面。请注意，这是一个异步功能，即：当排行榜页面被显示时，你的游戏将继续在后台运行。因此，你应该 小心使用这个功能，并确保暂停游戏，或只允许在不干扰游戏的地方显示这个功能。

语法。

实现_显示_领导板()

返回。

不适用

例子。

if achievement_available()
{
global.Pause = true;
instance_create_layer(0, 0, "Controllers", obj_Pause);
achievement_show_leaderboards() 。
}

上面的代码检查所选择的排行榜和成就服务是否可用，如果可用，它将设置一个全局变量并创建一个控制器 "暂停 "实例，然后调用函数在屏幕上显示排行榜。

返回。成就和排行榜
