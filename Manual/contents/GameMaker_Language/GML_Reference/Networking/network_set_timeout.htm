network_set_timeout

通过这个函数，你可以设置通过给定的套接字向/从服务器读写数据的超时。注意，超时不会产生任何类型的事件，所以你需要自己用报警器（例如）来处理超时。 注意，这个值只影响数据的发送和接收，如果你想改变连接超时值，那么你应该使用函数 network_set_config().

语法。

network_set_timeout(socket, read_timeout, write_timeout)。

争论 描述
插座 要使用的插座的ID。
read_timeout 来自服务器的传输将超时的毫秒。
Write_timeout 向服务器传输时的超时时间的毫秒。

返回。

不适用

例子。

network_set_timeout(sock, 3000, 3000)。

上面的代码将把通过变量 "sock "中索引的套接字读写数据的超时设置为3秒。
