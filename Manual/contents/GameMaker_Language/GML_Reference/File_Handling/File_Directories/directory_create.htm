目录_创建

该函数将在保存区创建一个具有给定名称的目录。

注意：这个功能在HTML5上不起作用，因为你不能在浏览器本地存储中创建或删除目录。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

语法。

directory_create(dname)

争论 描述
命名 要创建的目录的名称。

返回。

不适用

例子。

if !directory_exists("Games")
{
directory_create("Games")。
}

这将检查指定的目录是否存在于本地数据文件夹中，如果不存在，它会为你创建它。
