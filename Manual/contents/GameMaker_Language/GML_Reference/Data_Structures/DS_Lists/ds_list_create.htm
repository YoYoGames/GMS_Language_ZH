ds_list_create

这个函数将创建一个新的列表数据结构并返回索引值。这个值应该存储在一个变量中，并在所有与列表有关的后续函数调用中使用。请注意，如果你试图在列表被填充之前访问其中的一个值--即列表是 "空 "的--那么返回值可能是 0，因为在内部，列表中的前几个条目在创建时被设置，以减少最初向列表添加项目时的性能问题（尽管 ds_list_size()函数仍然会在一个新创建的列表中返回 0）。)如果你希望确保列表在创建时是 "真正的 "空的，那么你应该在创建列表后调用 ds_list_clear() ，这将意味着任何未填充的列表槽的返回值都是未定义的。

重要的是!当你创建一个数据结构时，识别它的索引值是一个从0开始的整数值。 这意味着不同类型的数据结构可以有相同的索引值，所以如果有疑问，你应该使用 ds_exists()函数来访问它们。还要注意的是，索引是可以重复使用的，所以一个被破坏的数据结构索引值可能会被之后新创建的数据结构使用。

语法。

ds_list_create()。

返回。

真实的

例子。

list = ds_list_create()。

这将创建一个新的列表，并将其索引ID分配给实例变量 "list"。
