ds_list_insert

这个函数将把给定的值添加到列表的给定位置。如果列表在给定位置之后包含更多的值，它们的位置将被上移一个以腾出空间，使列表大一个。

语法。

ds_list_insert(id, pos, val)。

争论 描述
身份证 要加入的列表的ID。
姿势 添加值的位置，其中0对应于列表的最开始，最终位置是ds_list_size(id)-1。
价值 要添加到列表中的值。

返回。

不适用

例子。

ds_list_insert(list, 9, score)。

上述代码将把存储在变量 "score "中的值添加到由变量 "list "索引的列表的第10个位置（列表从0开始计算，所以一个10个值的列表的编号是0-9）。

返回。DS列表
