ds_list_delete

通过这个函数，你可以删除存储在列表中某个特定位置的值。

语法。

ds_list_delete(id, pos)。

争论 描述
身份证 要改变的列表的ID。
姿势 在列表中的哪个位置删除该值。

返回。

不适用

例子。

if (ds_list_size(sc_list) > 10)
{
while (ds_list_size(sc_list) > 10)
{
ds_list_delete(sc_list, 0);
}
}

上面的代码检查DS列表的大小，如果大于10，它就在列表中循环，删除最上面的值（位置0），直到列表中只有10个条目。

返回。DS列表
