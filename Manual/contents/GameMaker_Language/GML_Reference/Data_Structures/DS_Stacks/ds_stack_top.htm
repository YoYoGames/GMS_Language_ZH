ds_stack_top

这个函数将只读取堆栈的第一个值（即 "在上面的"）。它不会弹出该值，这意味着它仍然可以在将来被这个函数或 ds_stack_pop(). 如果堆栈是空的，那么该函数将返回常数undefined，否则它将返回堆栈中包含的实数或字符串值。

语法。

ds_stack_top(id)。

争论 描述
身份证 要读取的数据结构的ID。

返回。

存储在堆栈中的数据类型值或未定义的数据类型值

例子。

num = ds_stack_top(control_stack)。

上述代码将从变量 "control_stack "索引的堆栈中读取数值，并将返回值存储在变量 "num "中。
