ds_map_replace

通过这个函数，你可以改变DS地图中给定键的值。你提供地图的索引（如函数所返回的 ds_map_create()) 然后提供要替换的键--可以是一个字符串或一个整数--以及要替换的键值。如果给定的键不 存在，那么它将为你创建，如果它存在，那么当前值将被新的 值替换。如果键值存在并且被替换，该函数将返回 true ；如果键值不存在并且创建了一个新的键值，则返回 false 。

语法。

ds_map_replace( id, key, val ) 。

争论 描述
身份证 要改变的地图的ID。
钥匙 含有应被新值替换的键。
价值 用新的值来替换给定的值

返回。

布尔型

例子。

ds_map_replace(inventory, "torso", 55)。

上面的代码在DS地图中查找键 "torso"，当它找到它时（如果不存在，则创建它），当前值被替换成指定的值。

返回。DS地图
