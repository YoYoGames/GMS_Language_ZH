os_get_language

这个函数返回一个字符串，包含运行游戏的操作系统的两个字母的语言代码，由ISO639标准设定。如果该信息不可用，它将持有 只是一个空字符串""，或 "en "代表 "英语 "语言。请注意，有些语言也有相关的区域代码，所以要区分同一国家的不同地区，请使用函数 os_get_region().

下表显示了ISO 639所定义的一些主要的双字母语言代码的例子。

语言 编码

注意：这不是返回的位置国家代码，而是操作系统的语言代码。

语法。

os_get_language()

返回。

字符串

例子。

switch (os_get_language())
{
case "es": ini_open("spanish.ini"); break;
case "fr": ini_open("french.ini"); break;
case "it": ini_open("italian.ini"); break;
default: ini_open("English.ini"); break;
;

上述代码检查操作系统的语言，并根据返回值打开一个不同的*.ini文件。
