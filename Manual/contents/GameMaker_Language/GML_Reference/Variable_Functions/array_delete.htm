array_delete

通过这个函数，你可以从一个数组的任何给定位置删除一个（或多个）值。该函数要求你提供一个保存数组的变量，要删除的数组中的索引（位置），以及要删除的值的数量。

语法。

array_delete(variable, index, number)。

争论 描述
变量 保存数组的变量。
指数 要删除该值的数组中的索引（位置）。
数 要删除的值的数量。

返回。

不适用

例子。

array_delete(score_array, 0, 10);

上面的代码将删除给定数组中从索引0开始的前10个值（所以索引0到9将被从数组中删除）。

返回。可变的功能
