变量_实例_名称_计数

通过这个函数，你可以找到为一个实例定义的变量的总数量。你提供要检查的实例ID，该函数将返回所遇到的变量数量的整数，或者（如果不存在给定ID的实例 ID的实例不存在）-1。

语法。

variable_instance_names_count(instance_id)。

争论 描述
instance_id 要检查的实例的唯一ID值。

返回。

整数

例子。

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

上面的代码将检索给定实例中的变量数量，并在控制台输出中显示该值的调试信息。

返回。可变的功能
