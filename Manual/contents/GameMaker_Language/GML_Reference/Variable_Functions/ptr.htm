这个函数将尝试把一个给定的值转换成一个指针数据类型，这个值必须是一个 real 、一个 string 、一个 int64 、一个 int32 。 或 ptr 。其他的都会导致游戏崩溃并出现错误信息。

语法。

ptr(n)。

争论 描述
n 要转换的值。

返回。

指针

例子。

if !is_ptr(val)
{
val = ptr(application_surface);
}

上面的代码检查变量 "val "是否包含一个指针，如果不包含，就给它分配一个指针。

返回。可变的功能
