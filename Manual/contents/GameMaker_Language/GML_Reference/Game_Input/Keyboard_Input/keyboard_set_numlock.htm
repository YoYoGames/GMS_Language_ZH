keyboard_set_numlock

你可以用这个功能来打开或关闭键盘数字锁（设置为 "true"表示打开，"false"表示关闭）。

注意： 该功能仅在Windows目标构建中可用，在任何其他设备上都不会发挥作用。

语法。

keyboard_set_numlock(value)。

争论 描述
价值 设置为 "true "表示 "开启"，"false "表示 "关闭"

返回。

不适用

例子。

如果 keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
else
{
keyboard_set_numlock(true);
}

上面的示例代码将获得数字锁键的状态，如果它是开（真），它将把它设置为关（假），反之亦然。

返回。键盘输入
