日期_比较时间

通过这个函数，你可以检查两个日期和时间，看哪一个的时间成分比另一个早或晚。如果datetime1较早，该函数返回-1，如果datetime1较晚，则返回1，如果它们相同，则返回0，并且忽略了日期。 因此，从字面上看，只是哪个时间比它给定的日期更远。

语法。

date_compare_time(datetime1, datetime2)。

争论 描述
日期时间1 第一个日期时间。
日期2 要与第一个日期比较的日期时间。

返回。

真实的

例子。

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime( ))。

这将把 "d "设置为相应的值，取决于哪个时间是最早的。基本上，如果当前时间晚于上午11:04，它将返回1。如果时间是11:04，它将返回0。

返回。日期和时间
