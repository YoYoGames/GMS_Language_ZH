随机获取种子

为了生成一个随机数，GameMaker Studio 2以一个随机种子数开始。通过这个函数，你可以检索到这个种子，然后可以存储它来重现一系列特定的随机事件。例如，假设你有一个程序性的 生成的游戏，但希望玩家能够回到以前发现的区域。那么，在每个区域的开始，你可以用这个函数存储随机种子，所以每次你需要回到那里，你可以用 random_set_seed() 将种子设置为以前的样子，从而迫使GameMaker Studio 2重现它以前使用的同一套 "随机 "函数。

语法。

random_get_seed()。

返回。

真实的

例子。

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed() );
ini_close();

上述代码将获得随机种子并将其存储在ini文件中。
