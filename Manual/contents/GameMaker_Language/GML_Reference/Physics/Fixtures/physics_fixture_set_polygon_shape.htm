物理学_夹具_设置多边形_形状

这个函数为你的灯具设置一个多边形形状，但是你需要用 physics_fixture_add_point()来实际定义这个多边形相对于灯具原点的形状。该多边形 在灯具被绑定到一个实例时被关闭。你也应该注意，这个函数必须在定义任何点之前被调用，而且你必须在将多边形绑定到一个实例之前为其定义至少三个点，否则 否则你会得到一个错误。

语法。

physics_fixture_set_polygon_shape(fixture)

争论 描述
固定设备 夹具的索引

返回。

不适用

例子。

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100)。

上面的代码将对变量 "fix_Ship "中索引的夹具应用一个多边形形状，然后定义三个点来创建一个三角形形状。
