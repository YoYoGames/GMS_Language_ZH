物理学_夹具_设置摩擦力

摩擦力是阻止材料元素相互滑动的相对运动的力量，在GameMaker Studio 2的物理学世界中，它被翻译成两个实例与固定装置的碰撞所造成的动量损失。 所造成的动量损失。因此，当两个实例发生碰撞时，它们的运动会受到这个值的影响，高的摩擦力会比低的摩擦力造成更大的动量损失。请注意，摩擦力通常被设置为0到1之间的值，但是如果需要，你可以使用任何非负的 值，如果需要的话。

语法。

physics_fixture_set_friction(fixture, friction)

争论 描述
固定设备 夹具的索引
摩擦力 夹具的摩擦力

返回。

不适用

例子。

physics_fixture_set_friction(fix_Brick, 0.1)。

上面的代码将把 "fix_brick "中索引的夹具的摩擦力设置为0.1。
