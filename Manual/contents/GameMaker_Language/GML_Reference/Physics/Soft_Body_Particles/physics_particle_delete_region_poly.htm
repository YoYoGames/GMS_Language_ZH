物理学_粒子_删除_区域_保利

通过这个函数，你可以从当前房间的物理模拟中删除（移除）所有属于定义的多边形区域范围内的粒子。该函数接收一个（先前创建的） ds_list() 包含多边形每个点的x/y位置，列表中的偶数位置是x坐标，奇数位置是y坐标，即：对于一个三角形，你的列表将有六个条目，其中0、2和 4为x坐标，1、3、5为y坐标。然后，多边形将被用来删除所有落在定义区域内的粒子。

请注意，所定义的多边形必须至少有三个点，最多有八个点。

语法。

physics_particle_delete_region_poly(pointList)

争论 描述
点列表 用来创建多边形的点的ds_list。

返回。

不适用

例子。

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i +;)
{
ds_list_add(p_list, mx[i]);
ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly（p_list）。
ds_list_destroy(p_list)。

上述代码将删除在由添加到DS列表的点所定义的多边形区域中发现的所有粒子。
