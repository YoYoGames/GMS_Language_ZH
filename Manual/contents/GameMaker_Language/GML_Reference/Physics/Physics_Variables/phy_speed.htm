phy_speed

这个只读变量返回启用物理的实例的当前速度，定义为每步像素。如果你需要改变这个值，你必须通过改变X和Y向量来实现，使用变量 phy_speed_x 和phy_speed_y来改变。

语法。

phy_speed;

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if phy_speed > 10
{
phy_linear_damping += 0.01;
}
else
{
phy_linear_damping = 2;
}

上面的代码检查物理学启用实例的速度，然后根据返回值改变线性阻尼。
