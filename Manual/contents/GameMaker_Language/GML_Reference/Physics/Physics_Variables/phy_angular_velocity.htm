phy_angular_velocity

这个变量可以用来设置实例的角速度，也可以用来获取当前的角速度，单位是度/秒，使用的值可以是正值（顺时针旋转）或负值（逆时针旋转）。 如果你在一个以前是静态的实例上设置这个值（即：它的密度为0），它将成为一个运动物体并开始旋转。

语法。

phy_angular_velocity;

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

如果abs（phy_angular_velocity）>0
{
phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
}
else
{
phy_angular_velocity = 0;
}

上面的代码将检查实例的角速度，如果它不是0，那么它将每一步增加（或减少）一个小量，直到该值为0。
