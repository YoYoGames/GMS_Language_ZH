phy_collision_y

这个只读数组返回在两个启用物理的实例之间的碰撞中检测到的所有点的y位置。

注意：这个变量只在启用了物理学的实例的碰撞事件中可用。

语法。

phy_collision_y。

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

for(var i = 0; i < phy_collision_points; i += 1;)
{
part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);
}

上述代码在两个启用物理的实例之间的碰撞的所有定义点上创建粒子。
