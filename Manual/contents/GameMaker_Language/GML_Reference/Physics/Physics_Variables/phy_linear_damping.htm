phy_linear_damping

这个变量可以用来设置实例的线性阻尼，也可以用来获取当前的线性阻尼。阻尼是启用了物理学的实例向前运动的 "阻力 "大小，较低的值 允许实例更快地移动和加速，而更高的值则使它需要更多的力量来推动。

语法。

phy_linear_damping;

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_linear_damping = 10;
}
else
{
phy_linear_damping = 3;
}

上述代码将检查调用实例和 "obj_Water "的实例之间的碰撞，并相应地改变线性阻尼。
