uwp_license_trial_version

这个函数可以用来检查游戏是否有试用许可证。如果是，该函数将返回true，否则返回false。

语法。

uwp_license_trial_version()。

返回。

布尔型

例子。

如果 uwp_license_trial_version()
{
global.LevelCap = 10;
}

上面的代码检查应用程序是否处于试用许可之下，如果是，它将设置一个全局变量。
