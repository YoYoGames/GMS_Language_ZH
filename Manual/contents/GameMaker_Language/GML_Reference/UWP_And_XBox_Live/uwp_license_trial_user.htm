uwp_license_trial_user

这个函数将返回当前正在使用的试用许可证的用户ID指针，如果用户当前没有激活或游戏当前不在试用模式中，它将返回pointer_null。

语法。

uwp_license_trial_user()。

返回。

指针

例子。

if gamepad_button_check_pressed(0, gp_start)
{
if uwp_license_trial_version()
{
var _uid = uwp_license_trial_user();
uwp_show_help(_uid);
}
}

上面的代码检查是否有按钮被按下，然后检查应用程序，看它是否在使用试用许可证运行。如果是，那么它就会获得许可的用户ID，并为该用户打开帮助文件。
