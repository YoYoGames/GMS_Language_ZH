窗口_有焦点

通过这个函数你可以轮询窗口（或标签）的状态，如果它失去了焦点，该函数将返回false，否则将返回true。在大多数情况下，你可以简单地使用 os_is_paused()函数 来测试，但在一些非常特殊的情况下（例如Chrome应用程序上的游戏），该函数将不会触发，在这种情况下，你应该使用这个函数来代替。

注意：这个功能只对HTML5、Windows和MacOS有效。

语法。

window_has_focus()。

返回。

布尔型

例子。

如果 !window_has_focus()
{
scr_PauseGame()。
}

上面的代码将检查游戏窗口是否处于焦点位置，如果该函数返回错误，将调用一个脚本。

返回。游戏窗口
