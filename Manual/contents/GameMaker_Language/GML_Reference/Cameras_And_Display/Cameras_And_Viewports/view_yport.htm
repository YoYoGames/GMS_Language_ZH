查看_yport

通过这个内置的数组，你可以获得或设置给定视口的y位置。视图端口是屏幕上绘制视图的区域，在任何时候你最多可以有8个活动的视图端口（数组的值从0到7，包括8个 端口）。)现在，GameMaker Studio 2的默认情况是，游戏窗口（或背景画布）与房间的大小相同，但是当你激活视图端口和摄像机时，这种行为就会改变，所有视图端口的边界框的总大小会被使用。因此，如果你在两个不同的位置有两个不同的视口，它们所覆盖的总面积决定了游戏窗口的大小。下面的图片说明了这一点。

如果你只有一个视口，设置视口的x或y值可以 有一些有趣的效果，但通常并不实用(见下图)，因此最好保持x和y位置为(0,0)。

语法。

view_yport[0 ... 7]。

返回。

真实的

例子。

view_xport[0] = 0;
view_yport[0] = 0。

上面的代码将view port[0]的位置重置为显示器的(0,0)位置（左上角）。
