摄像机_set_view_size

你可以使用这个函数来更新房间内的视图摄像机的大小。你给出唯一的摄像机ID值（由不同的 Camera_create()函数返回的），然后给出宽度和高度 (像素)来设置摄像机。

语法。

camera_set_view_size(camera_id, width, height)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
宽度 摄像机视图的宽度，单位是像素
高度 摄像机视图的高度，单位是像素

返回。

不适用

例子。

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0])。

上面的代码将为分配给view port[0]的摄像机设置视图摄像机的尺寸。
