gif_save_buffer

用这个函数你可以保存出一个GIF动画。你提供GIF的索引（如函数所返回的 gif_open())，该函数将返回一个包含GIF数据的1字节对齐的增长缓冲区。请注意，最后的 GIF数据将使用通用884调色板进行调色（更多信息见这里）。

语法。

gif_save_buffer(gif_index)。

争论 描述
gif_index 要保存的GIF的ID

返回。

整数（缓冲区ID）。

例子。

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
global.capture_buff = gif_save_buffer（gif_image）;
count = 0;
save_gif = false;
}
count++;
}

上面的代码将创建一个GIF图像文件，其中有30个从应用程序表面拍摄的帧，然后将其保存到一个缓冲区。
