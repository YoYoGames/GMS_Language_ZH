房间_获取摄像机

通过这个函数，你可以得到分配给当前房间以外的特定视图的摄像机的唯一索引ID。你给出要使用的房间，要使用的视图端口（从0到7），该函数将返回一个摄像机索引。

语法。

room_get_camera(rm, vind)。

争论 描述
rm 要得到的房间的索引。
vind 要获取摄像机的视口的索引。

返回。

真实(相机指数)

例子。

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
room_set_camera(rm_Game, 0, global.MainCam);
}

上面的代码将一个新创建的房间里的摄像机分配到视口[0]。
