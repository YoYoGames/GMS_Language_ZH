层_y

你可以使用这个函数来设置图层在当前范围内的y位置。你可以提供图层的ID（当你创建图层的时候，你可以用 layer_create())或图层名称（作为一个字符串 - 这将对性能产生影响），该函数将沿着房间的纵轴将图层移动给定的像素数。

语法。

layer_y(layer_id, y)

争论 描述
layer_id 要设置的图层的唯一ID值的y位置
y 在房间里的y位置，将该层设置为

返回。

不适用

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
layer_x(lay_id, 0);
layer_y(lay_id, 0);
}

上面的代码检查给定的图层位置，如果它没有被设置为（0，0），那么它就被设置为该位置。

返回。一般层的功能
