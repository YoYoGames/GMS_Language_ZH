对象_获取_斯普利特

这个函数将告诉你，你要检查的对象是否有一个精灵，如果有，它将返回该精灵的索引，如果没有，则返回-1。请注意，这不是一个实例函数!你可以有一个对象没有 而同一对象的一个实例可以有一个精灵，反之亦然，或者它们甚至可以有不同的精灵。你可以使用 "实例 "变量来设置一个单独的实例的精灵。 萌芽_索引 实例变量来设置单个实例的精灵。

语法。

object_get_sprite(obj)。

争论 描述
obj 要检查的对象的索引

返回。

真实（精灵资产ID）。

例子。

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
sprite_index = _spr;
}

上面的例子将检查实例的 sprite_index 和实例的 object_index 的精灵。如果它们不一样，那么它将为实例分配与对象索引相同的精灵。 对象的索引。

返回。物体
