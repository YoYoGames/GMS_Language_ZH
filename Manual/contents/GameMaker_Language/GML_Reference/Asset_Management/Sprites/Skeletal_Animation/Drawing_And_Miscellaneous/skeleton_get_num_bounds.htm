骨架上的数字边界（skeleton_get_num_bounds

这个函数将返回与分配给运行代码的实例的骨架动画精灵相关的边界盒的数量。然后，这可以和函数 skeleton_get_bounds()来检索每个边框的数据。 检索每个边框的数据。

语法。

skeleton_get_num_bounds()。

返回。

整数

例子。

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
{
var b_info = skeleton_get_bounds(i);
if b_info[0] > 0
{
var data = b_info[1] + " :";
for(var j = 0; j < b_info[0]; j++;)
{
data += " ( " + string(b_info[(j * 2) + 2])+ ", " + string(b_info[(j * 2) + 2 + 1])+ ")";
}
draw_text(20, yy, data);
yy += 20;
}
}

上述代码将循环浏览与当前分配的精灵相关的每个边界框，然后将这些数据作为字符串画到屏幕上。
