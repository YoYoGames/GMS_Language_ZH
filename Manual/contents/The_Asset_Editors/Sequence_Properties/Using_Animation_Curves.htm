  • 嵌入曲线。这些是嵌入在序列中的曲线，对它们所使用的参数轨道是唯一的。
  • 共享曲线。这些是独立于序列使用动画曲线编辑器创建的曲线。

我们将首先解释如何使用动画曲线资产，这只是创建一个新的曲线，在动画曲线编辑器中设置通道，然后将曲线资产拖到dopesheet中，将其应用到一个参数上 轨道。


你也可以使用鼠标右键 菜单选项Assign Shared Curve (Import) 来增加一个动画曲线资产 到选定的参数轨道上。这将打开资产资源管理器，你可以从呈现给你的曲线中选择所需的曲线。

值得注意的是，动画曲线资产必须有正确的通道数，因为每一种参数轨道都需要不同的通道数。 参数轨道的类型在动画曲线中都需要不同数量的通道。任何没有足够通道的动画曲线将在资产浏览器中显示 。

下面是不同的参数轨道和它们在每条曲线资产中所需要的通道的列表（通道应该按照每个参数类型的顺序添加）。

  • 位置：2个通道，X和Y。
  • 旋转。1个通道。
  • 颜色相乘。4个通道，Alpha、红、绿、蓝，范围必须在0到255之间。
  • 图像索引。1个通道。
  • 图像速度：1通道。
  • 原点：2个通道，X和Y。
  • 刻度：2个通道，X刻度和Y刻度，注意这个值是一个乘数，所以基准值是1（没有刻度），大于1会使图像变大，小于1会使图像变小。负值将翻转或 镜像图像，以及应用缩放。
  • 音高：1个通道，必须是最小的0，其中1是声音的默认音高。
  • 音量：1个通道，必须 最小为0，最大为1。

如果你导入的动画曲线没有足够的通道用于参数轨道，你会被提示选择是否要将任何缺失的通道添加到动画曲线中。

如果在外部动画曲线编辑器中删除了任何所需的通道，共享曲线也可以变得无效。在这种情况下，共享曲线会在参数表中显示为无效，你可以点击 "添加通道 "来创建任何缺少的通道，从而使曲线重新有效。 你可以点击 "添加通道 "来创建任何缺失的通道，这将使曲线再次有效。

默认情况下，动画曲线将沿着它所连接的资产键的整个长度应用，然而，就像关键帧一样，你可以在曲线上点击 ，然后拖动它来重新在时间线上定位，你也可以按住 键，然后拖动来调整曲线的长度。 (这也适用于嵌入式曲线)。

将参数曲线添加到轨道中的另一种方法是嵌入它。要做到这一点，可以在参数轨道上点击鼠标右键 ，选择"转换为曲线"，或者点击 "切换曲线模式 " 按钮，这样就可以在完整的涂料表区域内将该轨道显示为一条动画曲线，然后你就可以点击 "转换为曲线"按钮。

一旦你将参数关键帧轨道转换为曲线，就可以在dope sheet中进行编辑，与编辑动画曲线资产的方式完全一样。 这包括添加或删除点，改变它们的位置，改变曲线类型（线性、平滑或贝塞尔），应用预设，以及重命名通道和与之相关的颜色。 与它们相关的颜色。

重要提示：你也可以通过这种方式编辑已经应用到参数轨道上的动画曲线资产，但是你所做的任何修改都会应用到所有使用该动画曲线的轨道上，因为你是在编辑 曲线资产，而不是它的单一副本。要编辑一个只应用于某个轨道的动画曲线资产，你必须先在参数轨道上使用鼠标右键菜单 ，然后选择嵌入曲线。这将复制动画曲线资产，并将其嵌入到参数轨道中，作为仅适用于该轨道的独立曲线。反之，你也可以将嵌入的动画曲线 转换成动画曲线资产，使用鼠标右键 菜单选项Export To Shared Curve。这将把嵌入的 曲线转换成一个新的资产，然后可以像平常一样进行编辑，并在需要时应用到其他参数轨道。

转换控制

在这里，关键帧可以被转换成参数曲线（正如本页前面所解释的那样），但也有一个按钮可以将参数曲线转换成关键帧。然而，你应该知道，在这样做的时候，有几个注意事项。关键帧 不能在 "帧之间"，所以有些曲线值需要四舍五入到最接近的帧，这反过来又会改变参数轨迹的表现方式。另外，如果参数曲线使用catmull-rom或bezier插值，这将会丢失 ，而 关键帧之间的插值将被设置为线性插值。
