项目图像生成器

项目图像生成器可从GameMaker Studio 2 IDE顶部的工具菜单中进入。 从GameMaker Studio 2IDE顶部的工具菜单进入。这个工具旨在从一组图片中快速创建所有不同的闪屏、宣传图片和图标，这是所有不同目标平台所需要的。你提供基本图像 和基本图标，然后点击 "生成 "按钮，每个平台的 "游戏选项 "就会用新的图像进行更新。

为了使其正常工作，你需要提供一个高分辨率的闪屏和一个高分辨率的图标图像。推荐的尺寸是。

  • 飞溅屏幕：至少1920x1080px，或1080x1920px的纵向屏幕，并以24bit *.png 的形式编写，没有透明性。
  • 图标：至少1024x1024px（这是要求的最大图标的尺寸），并以24位 *.png ，可以有透明性。

一旦你提供了这些图像，它们将显示在预览窗口中，然后你可以设置将它们应用于不同平台时的 "适合 "类型。你可以从以下方面进行选择。

  • 保持长宽比 - 这将缩放图像以保持长宽比，如果不是1:1，则添加背景色以填充两侧。
  • 拉伸适应- 这将拉伸图像以适应图像/图标的完整尺寸。
  • 裁剪适合- 这将采取基本的图像，并使其最适合沿水平或垂直轴，然后裁剪其余部分。例如，如果你有一个640x480的图像，必须用作480x800的闪亮屏幕。 那么该图像将被放大以适应垂直轴（480->800），图像的其余部分将被裁剪掉。同样，如果你有一个1080x1920的图像，必须适合800x800，那么该图像将被缩小以适合水平轴 轴，其余部分被裁剪掉。
