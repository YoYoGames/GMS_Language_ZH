成就_登录_状态

这个函数目前只对Android平台上的Google Play有用。它将检查用户是否已经登录到谷歌服务，如果是，它将返回true，如果不是，则返回false。如果它返回 为真，就不需要再调用这个 achievement_login()函数。

语法。

achievement_login_status()。

返回。

布尔型

例子。

if !achievement_login_status()
{
achievement_login()。
}

上述代码将使你登录到所选择的排行榜和成就服务。

返回。成就和排行榜
