phy_col_normal_x

这个只读变量返回碰撞法线的x分量，对应于 phy_collision_x数组的值。对于每一个接触点，都有一个相关的接触法线（这通常是 碰撞中所有接触点的法线都是一样的）。)这个接触法线是一个单位向量，从碰撞中的一个实例指向另一个实例，例如，可以用来计算正确的 "推 "的方向以解决 碰撞

注意：这个变量只在启用了物理学的实例的碰撞事件中可用。

语法。

phy_col_normal_x。

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

repeat(5 + irandom(5))
{
with (instance_create_layer(x, y, "Effects", obj_Debris)
{
physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]) ;
}
}

上面的代码使用接触法线来设置在两个启用物理的实例之间的碰撞事件中创建的实例的运动方向。
