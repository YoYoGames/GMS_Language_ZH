园艺旋转

这个变量可以用来获取（或设置）实例灯具的角度，单位是度，类似于设置或获取. 图像_角度.但是要注意的是 在物理世界中，旋转的计算方式与普通GameMaker Studio 2游戏世界的计算方式相反，这意味着矢量函数如 point_direction()应该 等矢量函数的返回值会被修改（简单地将正数改为负数就可以解决这个问题）。

语法。

phy_rotation。

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y))/10;
}

上述代码检查线性速度，如果任一矢量不为0，它就会计算出实际速度，并使用该速度来设置旋转。
