物理学_粒子_设置阻尼

通过这个函数，你可以设置模拟中粒子的线性阻尼。阻尼是用来减少物理模拟中的实例随时间变化的速度，很像现实世界中的空气阻力。这个函数的范围是全局的。 因为它不仅会改变改变后产生的新粒子的阻尼，也会改变已经存在于模拟中的粒子的阻尼。

语法。

physics_particle_set_damping(damping)

争论 描述
减震 要应用于粒子夹具的线性阻尼。

返回。

不适用

例子。

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

上述代码将为模拟中的所有粒子设置基本属性。
