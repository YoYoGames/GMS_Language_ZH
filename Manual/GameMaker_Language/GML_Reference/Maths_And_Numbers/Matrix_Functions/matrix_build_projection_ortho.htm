矩阵_构建_投影_ortho

这个函数根据下面列出的指定参数建立一个正投影矩阵（这是你在GameMaker Studio 2中创建一个房间时使用的默认投影方式，不需要改变任何东西）。有时你需要将 从透视投影切换到正视投影，这就是这个函数的作用。它通常用于绘制一个覆盖图，例如，显示分数或其他方面，因为这提供了一个 "平面 "视图的元素绘制（即，没有透视）在3D游戏。 在3D游戏中没有透视）。请看下面的图片，了解正视图和透视图的区别。

注意：如果你希望不管当前的深度值如何，都能画出信息，你可能还需要暂时关闭隐藏面的移除。

语法。

matrix_build_projection_ortho(width, height, znear, zfar)。

争论 描述
w 投影的宽度。
h 投影的高度。
znear 靠近剪接平面。
zfar 远处的剪接平面。

返回。

矩阵指数

例子。

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat) ;
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了一个新的俯视矩阵和正视矩阵，将它们的ID存储在本地变量中，然后用它们来设置分配给view port[0]的摄像机的视图和投影矩阵。
