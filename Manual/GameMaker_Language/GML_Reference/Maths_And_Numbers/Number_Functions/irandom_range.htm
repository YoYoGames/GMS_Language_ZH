随机范围

这个函数类似于 random_range()相似，只是将整数值作为输入。你提供范围的低值和高值，该函数将返回一个在给定范围内（包括）的随机整数值。 包括）给定范围内的随机整数。例如，irandom_range(10, 35)将返回一个在10和35之间的整数。

与 irandom()函数一样，可以使用实数，在这种情况下，它们将被四舍五入到最接近的整数EG：irandom_range(6.2,9.9)将给出一个6到9之间的值。

注意：由于GameMaker.Studio每次重新运行游戏时，这个函数将返回相同的值。Studio每次都会生成相同的初始随机种子，以使调试代码成为一项更容易的任务。为了避免这种行为，请使用 randomise()在你的游戏开始时使用。

语法。

irandom_range(n1, n2)。

争论 描述
n1 将从中选择随机数的范围的低端。
n2 将从中选择随机数的范围的高端。

返回。

整数

例子。

score += irandom_range(500, 600);

这将在总分上增加一个500至600（含）之间的整数值。
