数学_获取epsilon

这个函数将返回目标平台的当前epsilon值。关于epsilon的更多信息，请参见函数 math_set_epsilon().

语法。

math_get_epsilon()。

返回。

真实的

例子。

var e = math_get_epsilon();
if e != 0.000001
{
math_set_epsilon(0.000001);
}

这将检索当前的epsilon值，并将其存储在一个本地变量中，然后对其进行检查，如果需要，将设置一个新的epsilon。
