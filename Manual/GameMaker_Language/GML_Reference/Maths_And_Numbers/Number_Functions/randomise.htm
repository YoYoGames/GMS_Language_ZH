随机

这个函数将种子设置为一个随机值。如果你需要在一个游戏的若干次运行中保持一个一致的值，你应该使用 random_set_seed().请注意，当在GameMaker Studio 2中使用随机数 函数时，初始种子总是相同的，因为这使得追踪错误和调试更加容易。如果你想用真随机进行测试，你应该在游戏开始时调用这个函数。该函数将 返回新的随机化种子值（一个无符号的32位整数）。

语法。

randomise();
// 或
randomize()。

返回。

整数（无符号的32位值

例子。

randomise()。
