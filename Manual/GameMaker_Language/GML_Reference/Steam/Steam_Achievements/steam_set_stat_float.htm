steam_set_stat_float

通过这个函数，你可以将一个特定的统计值设置为一个新的、浮动的值。这个统计值应该已经在你的游戏的Steamworks控制面板账户页面上定义过了，并且传递给这个函数的字符串应该是 匹配作为控制面板上的API名称。你可以用这个例子来追踪玩家走了多远，或者游戏完成的百分比。

语法。

steam_set_stat_float(stat_name, value)。

争论 描述
统计数据_名称 要设置的统计数据的名称（一个字符串）。
价值 要设置统计的值（32位浮点数）。

返回。

不适用

例子。

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc) 。

上面的代码根据旅行的距离变量 "dist "和你可以旅行的最大距离 "dist_max "计算出一个百分比，然后将 "Travelled "这个状态设置为新值。

返回。成就和统计数据
