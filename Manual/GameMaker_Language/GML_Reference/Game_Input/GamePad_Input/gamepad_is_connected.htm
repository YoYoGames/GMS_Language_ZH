gamepad_is_connected

这个函数将返回一个游戏板是否连接到给定的 "插槽"（返回true）或没有（返回false）。你通常会把这个函数和 gamepad_get_device_count() 函数来获取正确的可用游戏板和/或游戏板 "插槽 "的数量。请注意，在用户连接游戏板和GameMaker Studio 2检测到它被连接之间可能会有一点延迟（这在处理蓝牙连接的控制器时尤其如此）。 特别是在处理蓝牙连接的控制器时）。)

注意：在某些平台上--特别是游戏机--这个功能在连接/选择游戏手柄后可能立即返回错误，可能需要在几帧（步）后的警报中检查，以便正确检测到 游戏板。

语法。

gamepad_is_connected(numb)。

争论 描述
设备 要检查哪个游戏板 "插槽"。

返回。

布尔型

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

上面的代码在可用的游戏板（或游戏板插槽）中循环，然后检查每个游戏板的连接情况。然后用返回值将一个全局数组设置为真或假，以便在未来的检查中使用。

返回。游戏手柄输入
