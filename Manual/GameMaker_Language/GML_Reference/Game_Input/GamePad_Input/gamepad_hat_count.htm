游戏手柄_帽子数

这个函数将返回连接到给定设备 "插槽 "的游戏板的可用帽子总数。帽子一般指的是上/下/左/右按钮，注意在Windows目标上，帽子只对DirectInput 控制器（所以，从插槽4开始）。

语法。

gamepad_hat_count(device)。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。

返回。

真实的

例子。

h_num = gamepad_hat_count(4);

上述代码将返回插入设备 "插槽 "4的游戏板上可用的帽子数量，并将该值存储在变量 "h_num "中。

返回。游戏手柄输入
