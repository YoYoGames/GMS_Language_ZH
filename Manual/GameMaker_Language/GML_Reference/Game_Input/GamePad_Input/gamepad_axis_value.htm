gamepad_axis_value

您可以使用此功能从给定的游戏手柄插槽获取不同轴的值。您提供的手柄时隙索引来检查，与任一轴线索引常数沿（如列出这里），或0和之间的整数值 gamepad_axis_count()-1。对于每个可用的水平轴和垂直轴，返回值都将在-1和1之间。

请注意，返回值已标准化，以考虑为轴死区设置的值（使用函数 gamepad_set_axis_deadzone()），因此将始终在-1和0之间，以及0和1之间。例如，将死区设置为0.2表示向右推击摇杆只会在原始轴值结束时才开始返回从0-1的值。 0.2-因此，当原始值为0.2时，返回值为0；当原始值为0.5时，返回值为0.375；如果原始值为0.9，则返回值为0.875。

句法：

gamepad_axis_value(device, axisIndex);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
axisIndex要检查的轴索引（请参见常量列表）。

返回值：

real

例子：

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

上面的代码根据插入到设备“插槽” 0中的游戏手柄的左侧模拟控制器的返回值来设置实例的速度和方向。
