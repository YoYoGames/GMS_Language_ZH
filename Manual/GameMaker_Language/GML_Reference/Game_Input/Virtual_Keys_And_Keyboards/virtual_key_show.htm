虚拟键_显示

一旦你为你的设备创建了你的虚拟钥匙，你可能需要调试和测试它是否正确定位和正常工作。为此，你将调用这个函数，使用你想看到的虚拟键的索引（之前用 创建并存储的 virtual_key_add())，它将在屏幕上画一个彩色的矩形来代表该键。一旦你对事情感到满意，建议你创建并放置你自己的图形 在其区域内。

注意：被绘制的矩形将受到相应的绘制函数所设置的混合、颜色和阿尔法选项的影响。

语法

virtual_key_show(index)。

争论 描述
指数 要显示的虚拟钥匙的索引。

返回。

不适用

例子。

如果 global.Debug
{
virtual_key_show(global.Left)。
}
else
{
virtual_key_hide(global.Left);
}

上述代码检查全局变量 "global.Debug"，如果测试为真，那么变量 "global.Left "中索引的虚拟键将被画在屏幕上，如果测试为假，那么该键将被隐藏。
