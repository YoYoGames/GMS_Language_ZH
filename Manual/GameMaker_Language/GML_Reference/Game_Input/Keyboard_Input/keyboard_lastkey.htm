键盘_最后键

这个变量指的是 keyboard_key的值，返回该键的键码（所有标准的键码常数都被返回）。这个变量不是只读的，你可以 改变它，例如，如果你已经处理了它，把它设置为-1。

语法。

keyboard_lastkey;

返回。

真实的

例子。

if (keyboard_lastkey != -1)
{
str += keyboard_lastchar;
keyboard_lastkey = -1;
}

上面的代码检查lastkey变量是否不等于-1，如果是，就把最后一个键作为字符串添加到变量 "str "中，然后重置键盘_lastkey变量以接受进一步的输入。

返回。键盘输入
