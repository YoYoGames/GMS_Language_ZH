键盘_最后一个字符

这个变量存储了最后一次按下的键的字符串。这个变量不是只读的，你可以改变它，例如，如果你已经处理了它，可以把它设置为""（一个空字符串）。

语法。

keyboard_lastchar;

返回。

字符串

例子。

如果 keyboard_lastkey != -1
{
str += keyboard_lastchar;
keyboard_lastkey = -1;
}

上面的代码检查lastkey变量是否大于-1，如果是，就把最后一个键作为一个字符串添加到变量 "str "中。
