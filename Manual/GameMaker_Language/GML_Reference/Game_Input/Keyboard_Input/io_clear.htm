io_clear

这个函数将清除当前的键盘和鼠标状态。请注意，调用这个函数后，下一步会发生什么，取决于被检查的内容和被检查的平台。对于鼠标输入，鼠标按钮将 必须松开并再次按下，才能检测到任何进一步的输入，但对于键盘，就不那么简单了。一般来说，键盘将只在这一步被清除，但下一步将再次检测该键，然而 这在所有的平台和键盘上可能不一样，在某些情况下，该功能可能会像对鼠标那样，需要抬起并再次按下按键。

语法。

io_clear()。

返回。

不适用

例子。

io_clear()。

这将清除所有键盘和鼠标的状态。

返回。键盘输入
