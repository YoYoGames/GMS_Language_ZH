keyboard_get_numlock

你可以使用这个函数来查找键盘数字锁的状态，打开时返回true，关闭时返回false。

注意： 该功能仅在Windowsexe构建中可用，在任何其他设备上都不会发挥作用。

语法。

keyboard_get_numlock()。

返回。

布尔型

例子。

如果 keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
else
{
keyboard_set_numlock(true);
}

上面的示例代码将获得数字锁键的状态，如果它是开（真），它将把它设置为关（假），反之亦然。

返回。键盘输入
