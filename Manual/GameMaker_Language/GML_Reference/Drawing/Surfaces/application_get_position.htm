应用_获取位置

当你在目标平台的游戏选项中勾选了 "保持长宽比 "时，GameMakerStudio 2 将自动设置应用程序表面的绘制位置，以便在指定的显示器上正确地居中显示和缩放。 的中心位置和比例。然而，如果你正在操作这个表面，并希望自己绘制它，那么这个函数为你提供了一个简单的方法，可以准确地找到表面在显示器或窗口中被绘制的位置，这样你就可以 绘制它，或者将GUI图像或后期绘制的图像对准它。

该函数将返回一个有四个键的数组，其中键0和1是曲面左上角的x和y的位置，键2和3是曲面右下角的x和y的位置。 的位置，所有这些都是相对于显示器或窗口的尺寸而言的。

语法。

application_get_position()。

返回。

阵列

例子。

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1] 。

上面的代码将获得应用表面的位置，以及与显示窗口有关的绝对宽度和高度，并将它们存储在四个变量中供将来使用。
