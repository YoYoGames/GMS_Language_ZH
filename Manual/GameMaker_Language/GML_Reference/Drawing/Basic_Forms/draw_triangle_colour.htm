绘制三角形的颜色

通过这个函数，你可以画一个三角形的轮廓或一个填充的三角形。如果是填充，你可以为每个角点定义单独的颜色，如果这些颜色不一样，你将得到一个从一个到另一个的渐变效果 (颜色设置将覆盖用函数设置的基本颜色)。 draw_set_colour()).

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline) 。

争论 描述
x1 三角形的第一个角的X坐标。
y1 三角形的第一个角的y坐标。
x2 三角形第二个角的X坐标。
y2 三角形第二个角的y坐标。
x3 三角形第三个角的X坐标。
y3 三角形第三个角的y坐标。
col1 第一个角的颜色。
col2 第二个角的颜色。
col3 第三个角的颜色。
纲要 三角形是轮廓（真）还是填充（假）。

返回。

不适用

例子。

draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false) 。

这将画出一个填充的等腰直角三角形，直角处为红色，其他两个角为蓝色。
