draw_getpixel_ext

通过这个函数，你可以得到正在绘制到当前渲染目标的任何像素的完整的abgr 32bit值。这意味着结果将取决于调用该函数的事件，也取决于正在使用的目标表面。 使用的目标表面。

注意：这个功能会造成巨大的性能冲击，因此只有在绝对必要的时候才可以使用。

语法。

draw_getpixel_ext(x, y);

争论 描述
x 要检查的像素的X坐标
y 要检查的像素的y坐标

返回。

真实的

例子。

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

上面的代码将获得鼠标位置的32位颜色值，然后将其分割成各个组成部分的值，并将其存储在变量中。
