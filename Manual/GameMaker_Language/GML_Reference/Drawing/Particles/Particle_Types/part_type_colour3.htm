部件_类型_颜色3

这个函数可以用来为每个创建的给定类型的粒子设置三种颜色的梯度。第一种颜色是所有粒子开始时的颜色，第二种颜色是在其生命周期的一半时被混合的颜色 第三种颜色是粒子将结束的颜色。在粒子的生命周期中，各种颜色会发生平滑的梯度变化，从一种颜色到另一种颜色。

语法。

part_type_colour3( ind, color1, color2, color3 ) 。

争论 描述
ind 要改变的粒子类型的索引。
颜色1 粒子开始时的颜色。
颜色2 粒子在其寿命过半时将呈现的颜色。
颜色3 粒子结束时的颜色。

返回。

不适用

例子。

part_type_colour3( part_Health, c_red, c_white, c_maroon)。

上述代码将使所有以 "part_Health "变量索引的类型创建的粒子在其生命周期中具有从红色到白色到栗色的混合颜色。
