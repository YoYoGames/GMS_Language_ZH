粒子数

通过这个函数，你可以检查一个粒子系统当前是否有任何粒子在房间里被创建，它也会返回它们的数量。

语法。

part_particles_count(ind)。

争论 描述
ind 粒子系统的索引。

返回。

真实的

例子。

如果（part_particles_count(Sname) == 0）
{
part_system_destroy(Sname)。
instance_destroy()。
}

上述代码将检查变量 "Sname "中索引的本地粒子系统中的粒子数量，如果没有，它将销毁该系统，然后销毁实例。
