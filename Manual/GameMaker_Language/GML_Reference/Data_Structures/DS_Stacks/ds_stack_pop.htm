这个函数将从DS堆栈中弹出最高值，将其从堆栈中移除，并返回存储在一个变量中的值。如果堆栈是空的，那么该函数将返回常数undefined，否则它将返回 堆栈中包含的实数或字符串值。

语法。

ds_stack_pop(id)。

争论 描述
身份证 要弹出的数据结构的ID。

返回。

存储在堆栈中的数据类型值或未定义的数据类型值

例子。

if !ds_stack_empty(move_stack)
{
var xx = ds_stack_pop(move_stack);
var yy = ds_stack_pop(move_stack);
move_towards_point(xx, yy, 4);
}

上述代码检查变量 "move_stack "中索引的DS堆栈是否为空，如果不是，则从堆栈中弹出前两个值，用它们来设置移动方向。
