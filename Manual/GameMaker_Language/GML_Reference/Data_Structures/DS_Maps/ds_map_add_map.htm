ds_map_add_map

通过这个函数，你可以将一个（先前创建的）DS地图分配给给定DS地图中的一个键。这个函数是为创建JSON兼容的地图而设计的，然后你可以使用 json_encode() 编码的JSON兼容地图，并且只能与该功能一起使用。如果一个DS地图以这种方式添加了另一个地图，那么销毁父地图也会销毁所包含的地图并释放它们的内存，并且调用 ds_map_clear()在 也会销毁和清理任何有标记的地图。

语法。

ds_map_add_map(id, key, value)

争论 描述
身份证 要使用的地图的ID。
钥匙 添加地图的关键。
价值 要添加的地图的ID。

返回。

不适用

例子。

var j_map = ds_map_create（）;
var j_list = ds_list_create（）;
var sub_map = ds_map_create（）;
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map)。

上面的代码将创建两个DS地图，然后用一个列表和一个数组填充其中一个，再将其添加到第二个地图中，然后将其编码为一个JSON字符串。然后销毁地图，以删除它，以及它所包含的任何其他地图或列表。 从内存中删除。

返回。DS地图
