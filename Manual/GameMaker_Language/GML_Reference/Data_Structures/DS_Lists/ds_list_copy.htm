ds_list_copy

通过这个功能，您可以把一个列表的内容复制到另一个列表。两个列表都必须是以前创建的，如果被复制的列表中已经有信息，这个列表将被首先清除。最终的结果是两个独立的 列表，其中包含相同的信息。

语法。

ds_list_copy( id, source )。

争论 描述
身份证 被复制到的列表的ID。
来源 要复制的列表的ID。

返回。

不适用

例子。

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list) 。
}

上述代码将检查一个DS列表，看它是否为空。如果它不是空的，它就会被复制到另一个DS列表中（这个列表是以前创建的），然后原来的列表就被清除了。

返回。DS列表
