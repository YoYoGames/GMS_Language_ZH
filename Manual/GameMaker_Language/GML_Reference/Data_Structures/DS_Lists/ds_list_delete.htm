ds_list_delete

使用此功能，您可以删除存储在列表中特定位置的值。

句法：

ds_list_delete(id, pos);

争论描述
id要更改的列表的ID。
pos在列表中的何处删除值。

返回值：

N/A

例子：

if (ds_list_size(sc_list) > 10)
    {
    while (ds_list_size(sc_list) > 10)
        {
        ds_list_delete(sc_list, 0);
        }
    }

上面的代码检查DS列表的大小，如果它大于10，则会循环遍历该列表，删除顶部值（位置0），直到该列表只有10个条目。

返回： DS列表
