ds_grid_copy

通过这个函数，你可以将一个网格的内容复制到另一个网格。这两个网格必须是之前用 ds_grid_create()函数创建的。

语法。

ds_grid_copy(destination, source)。

争论 描述
目的地 要复制到的网格的这个索引。
来源 这个要复制的网格的索引。

返回。

不适用

例子。

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

上面的代码创建了一个新的DS网格，基于之前创建的网格的宽度和高度，然后将之前的网格信息复制到新的网格中。最后，它清除了旧的网格，使所有单元格的值为-1。
