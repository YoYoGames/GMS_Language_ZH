网络破坏

通过这个功能，你可以从游戏中删除一个网络插座连接。

注意：这个功能在运行在HTML5目标上的项目中使用时将不起作用。

语法。

network_destroy(socket)。

争论 描述
插座 要删除的插座的ID。

返回。

不适用

例子。

if !os_is_network_connected()
{
network_destroy(sock);
}

上面的代码将检查是否有一个数据连接，如果没有发现，就销毁ID为 "sock "的套接字。
