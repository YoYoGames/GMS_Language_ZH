淘宝网的用户显示名称是什么？

通过这个函数，你可以从给定的用户ID指针中检索出显示名称--作为一个字符串。请注意，如果用户是本地的，那么返回的值只是他们的显示名称的一个字符串，但如果是远程用户，那么这个字符串将是他们的游戏名。 是他们的游戏名，然后可以用于，例如，在语音聊天中静音/取消静音。

语法。

xboxlive_gamedisplayname_for_user（user_id）。

争论 描述
user_id 要检查的用户的ID指针。

返回。

字符串

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]) ;
}

上面的代码为每个可用的用户账户获取用户ID指针，然后将该指针与显示名称一起存储在两个数组中。

返回。用户和账户
