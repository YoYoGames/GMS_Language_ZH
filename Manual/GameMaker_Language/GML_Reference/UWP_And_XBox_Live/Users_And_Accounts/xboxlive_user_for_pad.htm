xboxlive_user_for_pad

这个函数将返回与给定的游戏手柄索引值相关的用户ID指针，如果没有用户与之相关，则返回指针_null。请注意，这个函数只能用于游戏手柄的索引，而不能用于 xboxlive_get_user_count().

语法。

xboxlive_user_for_pad(pad_index);

争论 描述
垫子指数 要检查的游戏板插槽的索引（一个整数）。

返回。

指针

例子。

for(var i = 0; i < 11; ++i;)
{
var u_id = xboxlive_user_for_pad(i);
if u_id != pointer_null
{
pad_name[i] = xboxlive_gamedisplayname_for_user（u_id）;
}
else
{
pad_name[i] = "Press Play";
}
}

上面的代码在所有可用的PAD槽中循环，并检查是否有一个与之相关的用户ID指针。如果有，一个数组被设置为用户显示名称，否则数组被设置为一些默认文本。

返回。用户和账户
