uwp_suspend

这个函数向系统表明，应用程序在检测到暂停请求后已经完成了数据保存。调用这个函数是Xbox One提交的一个要求，它应该总是与以下函数一起使用 uwp_is_suspending()一起使用，如下面的例子所示。

语法。

uwp_suspend()。

返回。

不适用

例子。

如果 uwp_is_suspending()
{
scr_Save_Game_Data()。
uwp_suspend()。
}

上面的代码检查应用程序是否进入暂停状态，如果是，则调用一个脚本，在暂停游戏之前保存游戏数据。
