uwp_was_closed_by_user

使用此功能，您可以确定游戏之前是否曾被用户关闭（即游戏已正确终止而不是系统终止）。如果已经存在，则该函数将返回 true，否则它将返回 false。

句法：

uwp_was_closed_by_user();

返回值：

Boolean

例子：

if uwp_was_closed_by_user()
    {
    global.GameLoaded = false;
    }

上面的代码检查该应用程序先前是否已被用户终止，如果已终止，则设置一个全局变量。

返回： UWP和XBox Live
