ini_key_delete

通过这个功能，你可以从ini文件中删除选定的键（及其相应的值）。

语法。

ini_key_delete(section, key)。

争论 描述
节 要删除一个键的部分。
钥匙 删除的关键。

返回。

不适用

例子。

ini_open("savedata.ini");
ini_write_real("save1", "Score",score);
ini_key_delete("save1", "Score");
ini_close()。

这个例子将打开 "savedata.ini "并向 "save1">"Score "写入一个值。然后它将删除 "Score "键并关闭.ini文件。
