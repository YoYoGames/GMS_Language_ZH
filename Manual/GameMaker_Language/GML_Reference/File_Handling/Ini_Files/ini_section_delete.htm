ini_section_delete

通过这个函数，你可以删除一个ini文件的整个部分，这也将删除与之相关的所有键值对。

语法。

ini_section_delete(section)。

争论 描述
节 要删除的部分。

返回。

不适用

例子。

ini_open("savedata.ini");
ini_write_real("save1", "Score", score ) ;
ini_section_delete("save1");
ini_close()。

这个例子将打开 "savedata.ini "并在 "save1">"Score "中写入一个值。然后它将删除 "save1 "部分并关闭.ini文件。
