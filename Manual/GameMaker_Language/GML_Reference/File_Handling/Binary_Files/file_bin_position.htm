file_bin_position

这个函数将返回给定文件ID的文件的当前位置（字节），其中0是第一个位置。你提供的是文件ID值，正如该函数所返回的那样 file_bin_open().

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_position(binfile)。

争论 描述
二进制文件 要获取位置的文件的ID。

返回。

真实的

例子。

pos = file_bin_position(file)。

这将在变量 "pos "中存储当前位置。

返回：二进制文件
