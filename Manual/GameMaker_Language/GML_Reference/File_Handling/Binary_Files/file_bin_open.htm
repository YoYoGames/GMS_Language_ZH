file_bin_open

这个函数将打开指定名称的二进制文件。模式表明可以对该文件做什么。

  • 0=阅读
  • 1 = 写作
  • 2 = 既读又写

当文件不存在时，它会被创建，该函数返回文件的ID，该ID必须在其他函数中使用。你可以同时打开多个文件（最多32个），但不要忘了在你用完后关闭它们。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_open(fname, mode)。

争论 描述
fname 要读取的文件的名称。
模式 指明可以对该文件做什么。

返回。

真实的

例子。

file = file_bin_open("myfile.bin", 2);

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 "file"。

返回：二进制文件
