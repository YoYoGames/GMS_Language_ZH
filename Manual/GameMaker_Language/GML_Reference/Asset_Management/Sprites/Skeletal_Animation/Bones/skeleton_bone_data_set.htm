骨架_骨架_数据集

你的骨骼动画是由一些 "骨头 "组成的，你将在你的动画程序中定义并给它们命名，这个函数可以用来在任何时候为命名的骨头设置某些数据。注意，这个数据是指 是指骨架的默认姿态，而不是正在绘制的当前姿态（为此，可以使用函数 skeleton_bone_state_set())，并且必须从一个先前创建的 DS地图来设置，该地图应该有以下键和它们的等值。

  • "x":骨骼相对于父骨骼的局部X位置。
  • "y":骨骼相对于父骨骼的局部Y位置。
  • "角度"。骨骼相对于父骨骼的局部旋转。
  • "xscale"。骨骼的局部水平比例。
  • "yscale"。骨骼的局部垂直比例。
  • "父"。父骨的名称（一个字符串）。

语法。

skeleton_bone_data_set(bone, map)。

争论 描述
骨骼 骨头的名称（字符串）。
地图 存储骨骼数据的（先前创建的）DS地图。

返回。

不适用

例子。

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy( bone_map) 。

上面的代码创建了一个DS地图，然后用名为 "头 "的骨骼的数据填充它。然后，它用一个新的值替换地图中的 "父 "骨键，并用新的数据集再次设置 "头 "骨。
