音频系统是否可用

这个函数可以用来检查并查看音频系统是否已经被初始化，或者音频上下文是否正在运行。在所有平台上，当音频引擎被初始化时，这个函数将在游戏开始后立即返回真，但在HTML5目标上除外。 在HTML5上，音频上下文的状态可以随时改变，这取决于用户的输入、正在使用的浏览器及其版本，所以这个函数可以用来检查和查看音频是否可以播放。如果该函数返回false（即。 音频上下文状态没有运行），那么只有未流的声音可以播放（但不保证，你应该假设没有音频可以播放），而如果它返回true，那么 所有的音频都会播放。

语法。

audio_system_is_available()。

返回。

布尔型

例子。

if audio_system_is_available()
{
if audio_is_paused(global.Music)
{
audio_resume_sound(global.Music)
}
else
{
if !audio_is_playing(global.Music)
{
global.Music = audio_play_sound(snd_Music, 0, true);
}
}
}
else
{
if audio_is_playing(global.Music)
{
audio_pause_sound(global.Music)。
}
}

上述代码将根据音频系统是否被初始化和可用，暂停/不暂停一个音轨。
