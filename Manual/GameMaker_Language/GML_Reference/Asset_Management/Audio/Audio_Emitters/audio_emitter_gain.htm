音频发射器增益

这个功能设置声音的最大增益（音量）。一个声音的感知音量可以根据下降值和它相对于听众的位置而改变。 但通过这个函数设置增益，全部音量将永远不会超过指定的增益值。下面的图片说明了当fall-off大于0时，增益是如何影响发射器的音量的。

这个函数将改变正在播放的声音的音量，以及随后通过给定的发射器播放的所有声音。 的声音。请注意，在某些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波或失真。

注意： 最终的音量也会受到全局音频增益的影响，而全局音频增益是由功能设置的。 audio_master_gain().

语法。

audio_emitter_gain(emitter, gain)。

争论 描述
发射器 要改变的发射器的索引。
收益 最大增益（默认为1）。

返回。

不适用

例子。

如果上升
{
gain += 0.05;
if gain >= 1 up = false;
}
else
{
gain += 0.05;
if gain <= 0 up = true;
}
