音频主增益

通过这个功能，你可以设置所有声音和音乐的全局音量的绝对值。它是基于从0（无声）到大于0的任何数值的线性比例，尽管通常你会认为最大音量是1。 都可以使用，但根据所使用的声音和被编译的平台，当声音被播放时，你可能会得到失真或削波。这个功能将影响所有在游戏中播放的声音和音乐的相对音量。

语法。

audio_master_gain(增益)。

争论 描述
收益 全局音量的值（0到1）。

返回。

不适用

例子。

if keyboard_check(vk_up)
{
if vol < 1 vol += 0.05;
audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
if vol > 0 vol -= 0.05;
audio_master_gain(vol);
}

上面的代码检查上下方向键的按压情况，然后增加或减少变量 "vol"。这个变量然后被用来设置游戏中所有声音和音乐的全局增益。
