音频_声音_获取增益

这个函数将返回给定声音的当前增益值。这个声音可以是一个从正在播放的单个声音的索引中引用的声音，这个索引已经被储存在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数，或者是资产浏览器中的一个实际的声音资产。增益通常以0到1的数值计算，但在一些平台上，你可以有一个大于1的增益，虽然1的数值被认为是 1的值被认为是 "全音量"，任何更大的值都可能引入音频削波。

语法。

audio_sound_get_gain(index)。

争论 描述
指数 要获得增益的声音的索引。

返回。

真实的

例子。

if audio_sound_get_gain(snd_Music) != 1
{
audio_sound_gain(snd_Music, 1, 0);
}

上述代码将改变从索引为 "snd_Music "的声音中播放的音频的增益，如果其增益不等于1。
