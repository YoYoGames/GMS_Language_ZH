路径_位置previous

这个变量可以用来获取或设置一个实例在上一步中沿其当前路径的位置，它是一个介于0和1之间的标准化值，即：0是路径的起始位置，1则是终点位置。它类似于 xprevious和 yprevious变量的工作方式相似，只是它是针对路径的。它可以用于 比如在有东西挡住的情况下暂时停止路径跟踪器（见下面的示例代码）。

语法。

path_positionprevious;

返回。

真实的

例子。

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if collision_circle(xx, yy, 16, obj_Player, false,true)
{
path_position = path_positionprevious;
}

上述代码检查实例前面的一个区域是否与对象 "obj_Player "发生碰撞，如果有，则将实例设置回它在当前路径上占据的前一个位置，正如函数所分配的那样 path_start().

返回。路径变量
