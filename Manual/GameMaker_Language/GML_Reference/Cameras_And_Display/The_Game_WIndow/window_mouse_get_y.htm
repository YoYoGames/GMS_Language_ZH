窗口_mouse_get_y

如果是HTML5游戏，通过这个函数你可以获得鼠标光标在浏览器中的y位置（像素），如果是Windows、Ubuntu（Linux）或macOS游戏，则可以获得鼠标光标在显示器中的y位置。

注意：关于常规的鼠标功能，请参见鼠标输入一节。

语法。

window_mouse_get_y()。

返回。

真实的

例子。

wy = window_mouse_get_y()。

上述代码将鼠标的当前y轴窗口位置存储在变量 "wy "中。

返回。游戏窗口
