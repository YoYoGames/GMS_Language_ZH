窗口_设置最小高度

这个函数可以用来为你的游戏设置一个最小的窗口高度。如果你在目标平台的游戏选项中启用了窗口调整选项，那么玩家可以将游戏窗口调整到他们希望的任何尺寸，然而通过使用这个函数 你可以将最小高度限制在你指定的尺寸。如果你希望回到默认行为（即：没有最小值），那么使用-1的值。

注意：该功能仅适用于 Windows, macOS和 Linux平台上。

语法。

window_set_min_height(height)。

争论 描述
高度 游戏窗口的最小高度（像素）。

返回。

不适用

例子。

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上面的代码将为游戏窗口设置最小和最大的宽度和高度。
