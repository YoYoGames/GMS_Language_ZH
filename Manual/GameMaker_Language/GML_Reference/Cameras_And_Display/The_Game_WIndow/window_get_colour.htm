窗口的颜色

该函数返回游戏窗口的背景颜色。这个颜色代表了游戏窗口中那些没有被任何视图占据的区域将使用的颜色。下面的图片说明了这一点。

上面的图片有两个视图，有两个视口，每个视口画在不同的位置。 这就把游戏窗口拉长以容纳两个端口，并在没有显示视图的地方使用窗口颜色给背景着色。

语法。

window_get_colour()。

返回。

不适用

例子。

如果 window_get_colour() != c_black
{
window_set_colour(c_black);
}

上面的代码将检查窗口的颜色，看它是否被设置为黑色，如果不是，就把它设置为黑色。
