查看摄像机

这个数组持有分配给给定视口的唯一的摄像机ID，可以设置为一个新的摄像机，也可以读取以获得当前的摄像机，如果没有分配摄像机，则返回-1。在一个房间里最多可以有8个视口（数组值从0到 7），并且可以通过简单地将此变量设置为摄像机的ID值（如由函数 camera_create()或 camera_create_view()). 如果你已经通过房间编辑器将摄像机设置到某个视口，这个变量将保持该视口的摄像机ID。

语法。

view_camera[0 ... 7];

返回。

摄像机索引，如果没有指定摄像机则为-1

例子。

如果view_camera[0] == -1
{
view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

上面的代码检查视图端口[0]是否有连接到它的摄像机，如果没有，就创建一个并设置到它。
