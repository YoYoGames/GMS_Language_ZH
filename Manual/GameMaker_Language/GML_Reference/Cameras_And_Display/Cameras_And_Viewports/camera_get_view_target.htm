摄像机_获取视图_目标

此函数可用于检索给定摄像机的跟踪目标。返回值可以是一个对象索引、一个实例ID，如果没有设置跟随目标，则返回值为-1。

语法。

camera_get_view_target(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

对象索引，实例ID，如果没有设置目标，则为-1。

例子。

var target = camera_get_view_target(view_camera[0]);
if target != obj_Player
{
camera_set_view_target(view_camera[0], obj_Player);
}

上面的代码检索了分配给view port[0]的摄像机的跟随目标，然后检查它是否与对象索引 "obj_Player "相匹配。如果不匹配，那么视图摄像机将被设置为跟随该对象的一个实例。
