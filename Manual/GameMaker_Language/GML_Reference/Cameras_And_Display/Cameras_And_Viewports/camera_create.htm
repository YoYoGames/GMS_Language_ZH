创建相机

此函数将创建一个 "空 "摄像机，即：一个没有定义视图或投影设置的摄像机。一旦你创建了摄像机，你就可以使用不同的矩阵函数来设置视图和投影矩阵。你也可以使用这个函数来创建摄像机，然后使用不同的camera_set_*函数来设置视图，但是请注意，你必须至少提供一个位置和视图尺寸。 否则生成的摄像机将是无效的，并且在应用于视图端口时产生奇怪的结果（如果你没有设置对象的跟随、边界或移动速度，这些将被设置为房间编辑器中的视图摄像机的默认值）。

请注意，摄像机和所有的动态资源一样，在创建时将占用内存。这意味着你需要注意不要不必要地创建相机，并通过调用函数销毁那些你不需要的相机 camera_destroy(). 这也意味着，如果分配给全局变量，摄像机将在各个房间之间持续存在（分配给实例或局部变量时也会持续存在，但你在更换房间后无法访问它们，因此会有内存泄漏），所以 这样你就可以--例如--创建一个新的摄像机，将其分配给一个全局变量，然后在每个房间开始时设置所需的视图和投影矩阵以及你需要使用的视图端口。

语法。

camera_create()。

返回。

真实（唯一的相机ID值）。

例子。

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat）。)

上面的代码创建了一个新的摄像机并将其ID分配给view[0]。然后，视图和投影矩阵被创建并存储在本地变量中，然后被用来设置摄像机。
