查看_set_xport

这个函数可以用来设置给定视口的X位置。你给出了视图端口的索引（从0到7）和要放置它的新位置。

语法。

view_set_xport(view_port, x)

争论 描述
视图_端口 目标视图端口（0-7）。
x 新的X位置

返回。

不适用

例子。

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

上面的代码将检查视图端口[0]的X位置，如果它不在需要的位置，就会被设置到该位置。
