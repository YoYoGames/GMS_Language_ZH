camera_get_proj_mat

此功能可用于检索投影矩阵。该函数返回矩阵ID值，然后可将其用于其他矩阵函数或设置其他摄像机的投影矩阵（使用 camera_set_proj_mat()）。

句法：

camera_get_proj_mat（camera_id）

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

Matrix ID (real)

例子：

mat = camera_get_proj_mat(camera_view[0]);

上面的代码获取分配给视口[0]的摄像机的投影矩阵。
