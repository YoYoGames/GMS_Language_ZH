查看_启用

这个变量控制是否启用房间内可见的任何视图端口。如果你把视口设置为可见，然后禁用这个选项，那么整个房间就会按照窗口大小的比例绘制到屏幕上，而不是通过视口绘制不同的摄像机。 而不是通过视口绘制不同的摄像机。

语法。

view_enabled。

返回。

布尔型（真：启用，假：禁用）。

例子。

if !view_enabled
{
view_visible[0] = true;
view_enabled = true;
}

上面的代码检查房间的视图是否被启用，如果没有，它将确保视图端口[0]是可见的，然后启用房间的视图。
