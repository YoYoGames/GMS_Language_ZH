Si la structure de données est vide Avec cette action, vous pouvez vérifier a pour voir si la structure de données donnée est vide ou non. Vous devez d'abord choisir le type de structure de données à vérifier (soit une liste, une carte ou une pile - pas de grilles car elles ne peuvent même pas être vides), puis vous fournissez la variable qui contient l'index de la structure de données (tel que renvoyé par les actions Créer une liste, Créer une carte et Créer une pile ). L'action reviendra true si la structure de données est vide, et false si ce n'est pas le cas. Notez que vous pouvez marquer "non" dans l'action pour vérifier si une structure de données contient des données en vérifiant si elle n'est pas vide. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if".
