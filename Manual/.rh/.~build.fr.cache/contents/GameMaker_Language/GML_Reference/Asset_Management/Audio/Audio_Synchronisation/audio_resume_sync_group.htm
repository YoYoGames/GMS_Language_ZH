audio_resume_sync_group Cette fonction reprendra le groupe de synchronisation donné s'il est en cours de lecture et a déjà été mis en pause (en utilisant la fonction audio_pause_sync_group ). L'index de groupe est la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group(). REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5. Syntaxe:audio_resume_sync_group(group_index); Argument Description group_index L'index de groupe à reprendre. Retour:N/A Exemple:if (keyboard_check_pressed((ord)"P"))    {    global.Pause = !global.Pause    if global.Pause       {       audio_pause_sync_group(sg);       }    else       {       audio_resume_sync_group(sg);       }    } Le code ci-dessus vérifie si une touche est enfoncée sur la touche "P", et si une est détectée, il bascule la variable "global.Pause" puis la vérifie pour mettre en pause ou reprendre le groupe de synchronisation indexé dans la variable "sg".
