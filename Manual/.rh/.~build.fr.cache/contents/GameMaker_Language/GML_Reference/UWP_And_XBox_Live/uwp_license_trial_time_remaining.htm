uwp_license_trial_time_remaining Cette fonction renverra le nombre approximatif de secondes jusqu'à l'expiration d'un essai basé sur le temps. La valeur renvoyée par cette fonction n'a pas de sens si l'essai n'est pas basé sur le temps ou si le jeu ne fonctionne pas en mode d'essai. Syntaxe:uwp_license_trial_time_remaining(); Retour:Real Exemple:if uwp_license_trial_version()     {     var secs = uwp_license_trial_time_remaining() mod 60;     var mins = uwp_license_trial_time_remaining() div 60;     var hours = mins div 60;     if secs < 10 secs = "0" + string(secs) else secs = string(secs);     if mins > 60 mins -= (hours * 60);     if mins < 10 mins = "0" + string(mins) else mins = string(mins);     if hours > 9 hours = "9" else hours = string(hours);     time_string = hours + ":" + mins + ":" + secs;     } Le code ci-dessus vérifie si le jeu est exécuté avec une licence d'essai et si c'est le cas, il crée une chaîne avec le temps jusqu'à l'expiration de la licence.
