is_int64 Cette fonction renvoie si une variable donnée est un entier 64 bits ou non. Dans certains cas, vous souhaitez vérifier et voir quel type de données contient une variable dans GameMaker Studio 2 et c'est à ce moment que vous utiliserez cette fonction. Il retourne true ou false selon que la valeur est un int64 ou non. Syntaxe:is_int64(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if !is_int64(val)     {     show_debug_message("Not a 64bit integer!");     } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un int64 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
