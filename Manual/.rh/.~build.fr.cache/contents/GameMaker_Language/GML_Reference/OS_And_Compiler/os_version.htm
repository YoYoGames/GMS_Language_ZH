os_version Cette variable vous indiquera le numéro de version du système d'exploitation qui exécute votre jeu. Par exemple, si vous l'exécutez sous Windows 10, os_version sera égal à 655360. Le tableau suivant présente les valeurs renvoyées pour les versions les plus récentes des différents systèmes d'exploitation: Système opérateur Nom de la version (numéro de version) Valeur de retour (exemples) Android Nougat (7,0 - 7,11) Oreo (8,0 - 8,11) Tarte (9,0) Android X (10,0) 24 - 25 26 - 27 28 29 iOS La valeur de retour est calculée comme suit: (major_version * 16777216) + (minor_version * 4096) + build_number iOS 10 (10,3) iOS 11 (11,4) iOS 12 (12,0) iOS 13 (13.0) iPhone iOS 13 (13.5) iPhone / Ipad 167784448 184565760 201326592 218103808 218124288 macOS X La valeur de retour est calculée comme suit: (major_version * 16777216) + (minor_version * 4096) + build_number El Capitan (10.11) Sierra (10.12) Haute Sierra (10,13) Mojave (10,14) Catalina (10,15) 167817216 167821312 167825408 167829504 167833600 Windows (la valeur de retour est calculée comme suit: majorVersion * 65536 + minorVersion) Windows 7 (6.1) Windows 8 (6,2) Windows 8.1 (6,3) Windows 10 (10.0) 393217 393218 393219 655360 REMARQUE: si vous avez besoin de plus d'informations sur le Windows vous pouvez utiliser la fonction environment_get_variable (). Syntaxe:os_version Retour:Real Exemple:if (os_type == os_android) && (os_version > 10)     {     global.GFX = 1;     } Le code ci-dessus vérifie le type de système d'exploitation et le numéro de version et ils sont tous les deux corrects, la variable globale est définie sur 1.
