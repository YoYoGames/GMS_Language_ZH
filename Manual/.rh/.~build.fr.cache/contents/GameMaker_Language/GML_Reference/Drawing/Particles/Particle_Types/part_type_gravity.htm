part_type_gravity Cette fonction définira la gravité qui affectera chaque particule du type donné qui est créée. La valeur de la force de gravité est ajoutée à la vitesse des particules à chaque pas et est généralement une petite valeur comme 0,5, tandis que la direction est la direction de la "traction" de gravité et suit les directions standard de GameMaker Studio 2 de 0 ° étant à droite, 90 ° étant vers le haut, 180 ° vers la gauche et 270 ° vers le bas. Syntaxe:part_type_gravity(ind, grav_amount, grav_direction); Argument Description ind L'indice du type de particule à modifier. grav_amount Force de la gravité. grav_direction La direction de la gravité. Retour:N/A Exemple:part_type_shape(global.p1, pt_shape_pixel); part_type_size(global.p1, 1, 3, 0, 0); part_type_scale(global.p1, 1, 1); part_type_colour1(global.p1, c_white); part_type_alpha2(global.p1, 1, 0); part_type_speed(global.p1, 2, 4, 0, 0); part_type_direction(global.p1, 0, 180, 0, 0); part_type_gravity(global.p1, 0.20, 270); part_type_orientation(global.p1, 0, 0, 0, 0, 1); part_type_blend(global.p1, 1); part_type_life(global.p1, 15, 60); Le code ci-dessus définira diverses valeurs de particules, y compris la gravité, ce qui ajoutera 0,2 à la vitesse à chaque pas avec une direction de 270 °, donc tirera la particule "vers le bas" vers le bas de l'écran.
