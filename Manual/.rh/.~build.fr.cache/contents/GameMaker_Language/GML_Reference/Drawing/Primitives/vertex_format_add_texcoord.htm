vertex_format_add_texcoord Dites à GameMaker Studio 2 d'accepter les données de position de texture (u et v) dans le cadre du nouveau format de sommet en cours de création. Syntaxe:vertex_format_add_textcoord(); Retour:N/A Exemple:vertex_format_begin(); vertex_format_add_textcoord(); vertex_format_add_position_3d(); my_format = vertex_format_end(); Le code ci-dessus créera un nouveau format de sommet avec juste des valeurs de texture et de position, puis stockera l'identifiant du format dans la variable "my_format".
