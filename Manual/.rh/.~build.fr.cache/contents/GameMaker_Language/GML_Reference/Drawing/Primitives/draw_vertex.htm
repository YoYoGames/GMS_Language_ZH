draw_vertex Cette fonction définit la position d'un sommet pour une primitive. L'aspect final de la primitive dépendra du type primitif choisi pour dessiner et de l'ordre dans lequel vous y ajoutez les sommets. Voir draw_primitive_begin() pour plus d'informations. Pour terminer et dessiner la primitive, vous devez appeler draw_primitive_end(). Syntaxe:draw_vertex(x, y) Argument Description x La coordonnée x du sommet. y La coordonnée y du sommet. Retour:N/A Exemple:draw_primitive_begin(pr_trianglelist); draw_vertex(100, 100); draw_vertex(100, 200); draw_vertex(150, 150); draw_primitive_end(); Le code ci-dessus dessinera une simple primitive de triangle.
