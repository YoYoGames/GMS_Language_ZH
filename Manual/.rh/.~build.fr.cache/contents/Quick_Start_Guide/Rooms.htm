Une salle est simplement un espace où vous placez des instances des objets qui composent votre jeu, ainsi que des tuiles et d'autres ressources graphiques. Lorsque vous créez une room pour la première fois dans l'Asset Browser, un nouvel espace de travail vous est présenté. Nous avons mentionné qu'il existe différents types de couches, alors passons en revue ce qu'ils sont et comment ils peuvent être utilisés lors de la création de vos jeux : Couche de fond Le calque d'arrière-plan est un calque qui peut être rempli d'une seule couleur ou d'une seule image. Il peut être déplacé et positionné dans la pièce, et vous pouvez avoir plusieurs couches d'arrière-plan. En général, il est utilisé, comme son nom l'indique, pour pour générer un arrière-plan constant pour tous les autres calques de la pièce. Par défaut, une nouvelle pièce contient toujours un calque d'arrière-plan, mais vous pouvez le supprimer si vous n'en avez pas besoin. Couche d'instance La couche d'instance est l'endroit où vous placez toutes les instances des objets dont vous avez besoin pour le jeu. Pour ajouter une instance à une couche, il suffit de cliquer sur sur l'objet dans le navigateur d'actifs et de le faire glisser dans l'espace de travail de l'éditeur de salle. navigateur d'actifs, puis faites-le glisser dans l'espace de travail de l'éditeur de salle. Par défaut, une nouvelle room contiendra toujours une couche d'instance, mais vous pouvez la supprimer si vous n'en avez pas besoin. Couche de carte en tuiles Après avoir créé un jeu de tuiles, vous devez ajouter les tuiles à votre pièce, ce qui se fait en créant une couche de carte de tuiles. Une couche de carte de carreaux est une couche qui vous permet d'ajouter des carreaux à partir de n'importe quel actif de jeu de carreaux que vous avez créé et qui sera automatiquement configurée pour utiliser une grille de la taille des cellules du jeu de carreaux. Vous ne pouvez ajouter qu'une seule tuile à chaque cellule de la grille, donc si vous avez besoin de plusieurs tuiles pour occuper le même espace, vous devez utiliser plusieurs couches de carte de tuiles. Couche du chemin La couche des chemins est la seule qui n'est pas réellement rendue lorsque vous exécutez votre jeu. Ce calque est plus un calque "pratique" pour vous aider à créer ou à modifier des actifs de chemin , puisqu'il vous permet d'ajouter ou de modifier des chemins dans l'espace réel de la pièce.
